## 引言
在纠错编码的世界中，[线性分组码](@entry_id:261819)是最基本也是最重要的一类。然而，在这些码中，有一类特殊的码——循环码——因其优美的[代数结构](@entry_id:137052)和极高的实现效率而脱颖而出。单纯地将码字视为向量，我们很难洞察其内在的对称性和规律性，这为高效的编码和解码设计带来了挑战。本文旨在填补这一认知鸿沟，通过引入[多项式代数](@entry_id:263635)的强大工具，系统地揭示循环码的奥秘。

在接下来的章节中，我们将首先在“原理与机制”中深入其核心，学习如何用多项式来表示码字，并理解[生成多项式](@entry_id:265173)是如何像一把钥匙一样构造出整个码空间。随后，我们将在“应用与跨学科联系”中见证这些理论的威力，看它们如何构建出[BCH码](@entry_id:268618)、[汉明码](@entry_id:276290)等著名纠错码，并延伸至[数据存储](@entry_id:141659)和前沿的量子信息领域。最后，通过一系列“动手实践”练习，您将有机会亲手应用所学知识，巩固对循环码工作方式的理解。

## 原理与机制

在信息论与编码理论中，循环码是一类具有丰富[代数结构](@entry_id:137052)和高效实现方式的[线性分组码](@entry_id:261819)。本章将深入探讨循环码的核心原理与工作机制，从其基本定义出发，逐步揭示其背后深刻的代数框架。我们将看到，通过将码字向量映射为多项式，循环码的性质变得清晰且易于操作，这不仅简化了编码和解码器的设计，也为构造性能强大的[纠错码](@entry_id:153794)（如[BCH码](@entry_id:268618)和[Reed-Solomon码](@entry_id:142231)）奠定了基础。

### 从向量到多项式：码字的[代数表示](@entry_id:143783)

[线性分组码](@entry_id:261819)中的一个码字通常被表示为一个长度为 $n$ 的向量 $\mathbf{c} = (c_0, c_1, \dots, c_{n-1})$，其中每个分量 $c_i$ 属于一个有限域，通常是二进制域 $\mathrm{GF}(2) = \{0, 1\}$。尽管这种[向量表示](@entry_id:166424)法直观，但它并未凸显循环码所特有的结构。为了揭示这种结构，我们引入一种等价的[代数表示](@entry_id:143783)法：将码字向量与一个次数小于 $n$ 的多项式一一对应。

具体而言，一个码字向量 $\mathbf{c} = (c_0, c_1, \dots, c_{n-1})$ 可以表示为码字多项式 $c(x)$：

$c(x) = c_0 + c_1 x + c_2 x^2 + \dots + c_{n-1} x^{n-1} = \sum_{i=0}^{n-1} c_i x^i$

在这个约定下，向量的第 $i$ 个分量 $c_i$（从索引0开始计数）是多项式中 $x^i$ 项的系数。这是一个至关重要的转换，因为它允许我们将[向量空间](@entry_id:151108)中的操作转化为多项式环中的代数运算。

例如，在一个码长为 $n=6$ 的[二进制码](@entry_id:266597)中，如果一个码字多项式为 $c(x) = x^4 + x^2 + x$，我们可以将其写为 $c(x) = 0 \cdot x^0 + 1 \cdot x^1 + 1 \cdot x^2 + 0 \cdot x^3 + 1 \cdot x^4 + 0 \cdot x^5$。根据系数与向量分量的对应关系，其[向量表示](@entry_id:166424)为 $(0, 1, 1, 0, 1, 0)$ 。

### 循环码的定义：[循环移位](@entry_id:177315)下的封闭性

循环码的名称源于其核心定义特征：**循环封闭性**。一个[线性分组码](@entry_id:261819)被称为**循环码**（Cyclic Code），如果其中任意一个码字的[循环移位](@entry_id:177315)（cyclic shift）仍然是该码中的一个码字。

对于一个码字向量 $\mathbf{c} = (c_0, c_1, \dots, c_{n-1})$，它的一次**右[循环移位](@entry_id:177315)**定义为向量 $(c_{n-1}, c_0, c_1, \dots, c_{n-2})$，即最后一个元素移到最前面，其余元素依次向右平移。同样，一次**左[循环移位](@entry_id:177315)**定义为 $(c_1, c_2, \dots, c_{n-1}, c_0)$。如果一个码是循环码，那么对任意码字进行任意次数的[循环移位](@entry_id:177315)，其结果都必须仍在码集之内。

这个属性并非所有[线性码](@entry_id:261038)都具备。考虑一个码长为6的二[进制](@entry_id:634389)[线性码](@entry_id:261038) $C = \{(0,0,0,0,0,0), (1,1,0,1,0,0), (0,0,1,1,0,1), (1,1,1,0,0,1)\}$。我们来检验非零码字 $\mathbf{c} = (1,1,0,1,0,0)$。对其进行一次右[循环移位](@entry_id:177315)，得到向量 $(0, 1, 1, 0, 1, 0)$。通过与码集 $C$ 中的所有码字进行比较，我们发现这个新向量并不属于 $C$。因此，这个码 $C$ 不是一个循环码 。这个反例清晰地表明，循环封闭性是一个对码集施加的强约束，也正是这个约束赋予了循环码优美的[代数结构](@entry_id:137052)。

### 代数引擎：[商环](@entry_id:148632) $R_n = \mathrm{GF}(q)[x]/(x^n-1)$

多项式表示法的真正威力在于它能将[循环移位](@entry_id:177315)操作转化为简洁的代数运算。让我们探究[循环移位](@entry_id:177315)在多项式域中对应着什么。

考虑码字多项式 $c(x) = \sum_{i=0}^{n-1} c_i x^i$。我们用 $x$ 来乘这个多项式：
$x \cdot c(x) = c_0 x + c_1 x^2 + \dots + c_{n-2} x^{n-1} + c_{n-1} x^n$

这个新多项式的次数可能为 $n$，超出了我们对码字多项式（次数小于 $n$）的定义。为了处理这个问题，我们将所有多项式运算置于一个特定的[代数结构](@entry_id:137052)中：商环 $R_n = \mathrm{GF}(q)[x]/(x^n-1)$。在这个环中，所有多项式都对 $x^n-1$ 取模。这意味着我们有基本关系 $x^n \equiv 1 \pmod{x^n-1}$。（在二[进制](@entry_id:634389)域 $\mathrm{GF}(2)$ 中，由于 $1=-1$，这个关系等价于 $x^n+1 \equiv 0$）。

利用这个关系，我们可以将 $x \cdot c(x)$ 的结果进行化简：
$x \cdot c(x) \pmod{x^n-1} = c_{n-1} \cdot (1) + c_0 x + c_1 x^2 + \dots + c_{n-2} x^{n-1}$

这个新的多项式 $c'(x) = c_{n-1} + c_0 x + \dots + c_{n-2} x^{n-1}$ 对应的码字向量是 $(c_{n-1}, c_0, \dots, c_{n-2})$，这恰好是原向量 $\mathbf{c}$ 的一次右[循环移位](@entry_id:177315) 。

这个优美的结论是循环码理论的基石：**在[商环](@entry_id:148632) $R_n$ 中，对码字多项式乘以 $x$ 等价于对码字向量进行一次右[循环移位](@entry_id:177315)。**

因此，一个[线性码](@entry_id:261038)是循环码，当且仅当其码字多项式集合在 $R_n$ 环中对任意多项式的乘法是封闭的。在代数上，这样的集合被称为环的一个**理想**（Ideal）。由于 $R_n$ 是一个[主理想](@entry_id:152760)环，其中的任何理想都可以由单个多项式生成。这个生成元就是我们接下来要讨论的[生成多项式](@entry_id:265173)。

### [生成多项式](@entry_id:265173)：构造循环码的钥匙

由于循环码的码字多项式集合构成 $R_n$ 中的一个理想，这个理想必然可以由一个唯一的、首一的（monic）多项式 $g(x)$ 生成。这个 $g(x)$ 被称为循环码的**[生成多项式](@entry_id:265173)**（Generator Polynomial）。

一个循环码 $C$ 可以被完全定义为：
$C = \{ m(x)g(x) \mid m(x) \in \mathrm{GF}(q)[x] \text{ and } \deg(m(x)g(x))  n \}$

换言之，所有码字多项式都是[生成多项式](@entry_id:265173) $g(x)$ 的倍式。一个多项式 $c(x)$ 是一个有效码字，当且仅当它能被 $g(x)$ 整除 。例如，如果一个二[进制](@entry_id:634389)循环码的[生成多项式](@entry_id:265173)为 $g(x)=x^3+x+1$，那么多项式 $c(x)=x^4+x^2+x$ 是一个有效码字，因为在 $\mathrm{GF}(2)$ 上进行[多项式除法](@entry_id:151800)可以验证 $c(x) = x \cdot g(x)$，余数为0。

那么，什么样的多项式可以作为码长为 $n$ 的循环码的[生成多项式](@entry_id:265173)呢？$g(x)$ 生成的理想 $\{m(x)g(x)\}$ 必须是商环 $R_n = \mathrm{GF}(q)[x]/(x^n-1)$ 的一个理想。这要求 $g(x)$ 必须是多项式 $x^n-1$ 的一个因子。这是成为[生成多项式](@entry_id:265173)的充要条件。

**定理**：一个次数为 $r$ 的[首一多项式](@entry_id:152311) $g(x)$ 是码长为 $n$ 的循环码的[生成多项式](@entry_id:265173)，当且仅当 $g(x)$ 是 $x^n-1$ 在 $\mathrm{GF}(q)[x]$ 上的一个因子。

为了找到所有可能的[生成多项式](@entry_id:265173)，我们需要对 $x^n-1$ 进行[因式分解](@entry_id:150389)。例如，对于码长为 $n=7$ 的二进制循环码，我们需要在 $\mathrm{GF}(2)$ 上分解 $x^7-1$（等价于 $x^7+1$）。其不可约因式分解为：
$x^7+1 = (x+1)(x^3+x+1)(x^3+x^2+1)$

因此，任何这些不可约因子的乘积都可以作为[生成多项式](@entry_id:265173)。例如，$g(x)=x^3+x+1$ 是一个合法的[生成多项式](@entry_id:265173)，其生成的码就是著名的 $(7,4)$ [汉明码](@entry_id:276290)。同样，$g(x)=(x+1)(x^3+x+1)=x^4+x^3+x^2+1$ 也是一个合法的[生成多项式](@entry_id:265173)  。然而，$x^2+x+1$ 并不是 $x^7+1$ 的因子，因此它不能作为码长为7的循环码的[生成多项式](@entry_id:265173)。

### 循环码的参数：码长、维数与度数

循环码的参数与其[生成多项式](@entry_id:265173)密切相关。
- **码长 $n$**：码字向量的长度，由 $x^n-1$ 中的 $n$ 决定。
- **[生成多项式](@entry_id:265173)的度数 $r = \deg(g(x))$**：这个度数决定了码的冗余度。
- **码的维数 $k$**：表示码中独立信息位的数量，即可以编码的不同消息的数量为 $q^k$。

这三个参数之间存在一个简单的关系：$k = n - r$。

这个关系可以这样理解：一个码字 $c(x)$ 由消息多项式 $m(x)$ 和[生成多项式](@entry_id:265173) $g(x)$ 构造而成，即 $c(x) = m(x)g(x)$。为了保证码字多项式的次数小于 $n$，即 $\deg(c(x))  n$，我们必须有 $\deg(m(x)) + \deg(g(x))  n$。由于 $\deg(g(x)) = r$，这意味着 $\deg(m(x))  n-r$。因此，消息多项式 $m(x)$ 的最高次数可以是 $n-r-1$。这样一个多项式 $m(x) = m_0 + m_1 x + \dots + m_{n-r-1} x^{n-r-1}$ 拥有 $k = n-r$ 个可以自由选择的系数 $(m_0, \dots, m_{k-1})$。这 $k$ 个系数就对应着 $k$ 个信息位。

例如，对于一个码长 $n=15$、维数 $k=11$ 的二[进制](@entry_id:634389)循环码，其[生成多项式](@entry_id:265173)的度数必须是 $r = n-k = 15-11 = 4$ 。

### 校验多项式与对偶码

与[生成多项式](@entry_id:265173)相辅相成的概念是**校验多项式**（Parity-Check Polynomial），记为 $h(x)$。它与[生成多项式](@entry_id:265173) $g(x)$ 的关系定义为：
$g(x)h(x) = x^n - 1$

因此，$h(x) = (x^n-1)/g(x)$。校验多项式的度数为 $\deg(h(x)) = n - \deg(g(x)) = n - r = k$。校验多项式提供了一种验证一个接收到的多项式 $v(x)$ 是否为有效码字的便捷方法。一个多项式 $v(x)$ 是码字当且仅当 $v(x)h(x) \equiv 0 \pmod{x^n-1}$。

以 $(7,4)$ [汉明码](@entry_id:276290)为例，其[生成多项式](@entry_id:265173)为 $g(x) = x^3+x+1$。对应的校验多项式为：
$h(x) = \frac{x^7+1}{x^3+x+1} = (x+1)(x^3+x^2+1) = x^4+x^2+x+1$ 。

这个概念也与**对偶码**（Dual Code）紧密相连。一个码 $C$ 的对偶码 $C^{\perp}$ 是与 $C$ 中所有码字都正交的向量集合。循环码的一个重要性质是：**循环码的对偶码也是循环码**。

对偶码 $C^{\perp}$ 的[生成多项式](@entry_id:265173) $g^{\perp}(x)$ 与[原码](@entry_id:754817)的校验多项式 $h(x)$ 之间存在深刻联系。具体而言，$g^{\perp}(x)$ 是 $h(x)$ 的**倒反多项式**（reciprocal polynomial）经过[标准化](@entry_id:637219)后的结果。对于一个次数为 $d$ 的多项式 $p(x)$，其倒反多项式 $\tilde{p}(x)$ 定义为 $\tilde{p}(x) = x^d p(1/x)$。

对于码长为7，由 $g(x)=x^3+x+1$ 生成的码，我们已经知道其校验多项式为 $h(x)=x^4+x^2+x+1$。其对偶码的[生成多项式](@entry_id:265173) $g^{\perp}(x)$ 就是 $h(x)$ 的倒反多项式：
$g^{\perp}(x) = \tilde{h}(x) = x^4 h(1/x) = x^4((1/x)^4 + (1/x)^2 + (1/x) + 1) = 1 + x^2 + x^3 + x^4$
即 $g^{\perp}(x) = x^4+x^3+x^2+1$ 。

从更高等的视角看，循环码的结构可以通过其[生成多项式](@entry_id:265173)在某个扩域中的根来刻画。一个码长为 $n$ 的循环码 $C$ 可以由其**定义集**（defining set）$T$ 唯一确定，$T$ 是 $g(x)$ 的根在扩域 $\mathrm{GF}(q^m)$ 中的指数集合。其对偶码 $C^{\perp}$ 的定义集 $T_{\perp}$ 与 $T$ 的关系为 $T_{\perp} = -T^c \pmod{n}$，其中 $T^c$ 是 $T$ 在 $\{0, 1, \dots, n-1\}$ 中的补集，负号表示模 $n$ 的[加法逆元](@entry_id:151709)。这一关系在[BCH码](@entry_id:268618)等更高级码型的设计中扮演着核心角色 。

综上所述，循环码的原理与机制深深植根于[多项式代数](@entry_id:263635)理论。通过将码字向量转化为多项式，并将运算置于商环 $R_n$ 中，我们不仅为[循环移位](@entry_id:177315)找到了简洁的代数对应，还引出了[生成多项式](@entry_id:265173)这一核心工具，它决定了码的全部结构、参数和构造方法。
