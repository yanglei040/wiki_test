## 引言
[系统发育树](@entry_id:140506)是[计算生物学](@entry_id:146988)和演化研究中不可或缺的工具，它以直观的图形方式揭示了物种、基因乃至文化现象之间的历史联系。然而，要准确构建和解读这些[树状图](@entry_id:266792)，首先必须掌握一套精确而独特的术语。对于初学者而言，诸如“支系”、“[并系群](@entry_id:147411)”、“[最近共同祖先](@entry_id:136722)”等概念往往构成理解上的壁垒，导致对演化关系的误读。本文旨在系统性地扫清这一障碍，为读者提供一个关于系统发育树术语的全面指南。

在接下来的内容中，我们将分三个章节逐步深入。第一章“原理与机制”将详细拆解系统发育树的基本构成，阐明从节点、分支到根的各项定义，并辨析单系、并系与复系等关键类群划分。第二章“应用与跨学科联系”将展示这些核心概念如何超越生物学，在历史语言学、软件工程乃至[流行病学](@entry_id:141409)等多元领域中发挥作用，揭示[树状思维](@entry_id:172954)模型的普适性。最后，在“动手实践”部分，你将有机会通过具体问题来巩固和应用所学知识，检验自己对这些术语的掌握程度。通过本次学习，你将能够自信地解读复杂的演化假说，并运用系统发育的语言来精确描述和分析各种具有层级结构系统的演化历史。

## 原理与机制

本章旨在系统性地阐述[系统发育分析](@entry_id:172534)中的核心术语和基本原理。理解这些概念是正确构建、解读和应用系统发育树的基石。我们将从[树状图](@entry_id:266792)的基本构成元素开始，逐步深入到更复杂的拓扑结构、[类群](@entry_id:182524)划分和定量信息的解释中。

### 系统发育树的基本组成部分

系统发育树是一种图形化的假说，用于表示一组生物单元（如物种、基因或病毒株）之间的[演化关系](@entry_id:175708)。尽管形式多样，但所有系统发育树都由一些基本组件构成。

**末端节点（Tips）或[叶节点](@entry_id:266134)（Leaves）**
[系统发育树](@entry_id:140506)的末端，通常表示为树枝的终点，被称为**末端节点**或**叶节点**。这些节点代表了分析中包含的实际观测单元，也就是我们从中收集数据的实体。例如，在一项旨在研究五种新呼吸道病毒变体（P, Q, R, S, T）[演化关系](@entry_id:175708)的研究中，研究人员对每个变体的刺突蛋白基因进行了测序，并构建了一棵系统发育树。在这棵树中，代表变体P, Q, R, S, T的五个点就是末端节点。它们代表了用于构建树的实际、可观测的病毒样本，而非任何理论上的祖先 。

**内部节点（Internal Nodes）**
连接分支的[交叉点](@entry_id:147634)被称为**节点**，特别是那些不位于末端的节点，我们称之为**内部节点**。每一个内部节点在生物学上都有着深刻的含义：它代表其所有后代谱系的一个**假定的[共同祖先](@entry_id:175919)（hypothetical common ancestor）**。这个祖先是一个在过去某个时间点发生分化的种群。例如，在分析五种未知真菌（A, B, C, D, E）的演化历史时，树上连接谱系A和谱系B的那个分叉点，即内部节点，就代表了真菌A和真菌B的**[最近共同祖先](@entry_id:136722)（most recent common ancestor, MRCA）**。这个节点所代表的祖先种群是真实存在过的，但通常是根据现有数据推断出来的，而非直接观测到的 。

**分支（Branches）或边（Edges）**
连接各个节点的线段被称为**分支**或**边**。每一条分支都代表一个**演化谱系（evolutionary lineage）**。分支的意义取决于树的类型。在最简单的树中，分支仅表示连接关系；而在更复杂的树中，分支的长度可以携带定量信息，例如演化时间或遗传变异量。

### 树根的意义：从[无根树](@entry_id:199885)到[有根树](@entry_id:266860)

[系统发育树](@entry_id:140506)可以分为两大类：**[无根树](@entry_id:199885)（unrooted trees）**和**[有根树](@entry_id:266860)（rooted trees）**。这个区别对于正确解读演化历史至关重要。

一棵**[无根树](@entry_id:199885)**只展示了[类群](@entry_id:182524)之间的相关性或距离，但没有指定演化的时间方向。它描绘了哪些类群是近亲，但没有说明哪个是“更古老的”或“更衍生的”。

相比之下，**[有根树](@entry_id:266860)**包含一个特殊的节点——**根（root）**，它代表了树中所有类群的[最近共同祖先](@entry_id:136722)。根的存在为树赋予了时间维度，从而确立了**祖先-后代关系（ancestor-descendant relationships）**。一旦树被“生根”，我们就可以明确地追踪从祖先到后代的演化路径。

这个时间方向的确立至关重要，因为它允许我们**极化（polarize）**性状的演化。例如，假设我们有一个包含四个物种的内群（ingroup），其某个性状的状态分别为{Alpha: 0, Beta: 0, Gamma: 1, Delta: 1}，拓扑结构为((Alpha, Beta), (Gamma, Delta))。在[无根树](@entry_id:199885)上，根据[最大简约法](@entry_id:168212)（maximum parsimony），我们只能推断出连接{Alpha, Beta}分支和{Gamma, Delta}分支的内部支上发生了一次状态变化，但无法确定变化的方向是 $0 \to 1$ 还是 $1 \to 0$。然而，如果我们引入一个已确定位于内群之外的**外群（outgroup）**，其[性状状态](@entry_id:151081)为 $0$，并用它来确定根的位置，那么我们就可以推断出内群的[共同祖先](@entry_id:175919)状态很可能为 $0$。这样，我们可以明确地推断出，在朝向{Gamma, Delta}的谱系上发生了一次从 $0$ 到 $1$ 的定向变化。因此，生根不仅定义了祖先和后代，还允许我们推断演化变化的方向 。

值得注意的是，外群生根法依赖一个核心假设：所选外群确实在所有内群成员分化之前就已经分离出去。如果这个假设不成立（即外群实际上嵌套在内群之中），那么所推断的根位置将是错误的，这将导致对祖先-后代关系和[性状演化](@entry_id:165250)方向的系统性误判 。

### [单系性](@entry_id:174362)与[类群](@entry_id:182524)划分

在现代演化生物学中，根据与[共同祖先](@entry_id:175919)的关系，生物[类群](@entry_id:182524)被划分为三种类型：[单系群](@entry_id:142386)、[并系群](@entry_id:147411)和复系群。

**[单系群](@entry_id:142386)（Monophyletic Group）或[支系](@entry_id:171685)（Clade）**
一个**[单系群](@entry_id:142386)**，或称**支系**，由一个共同祖先及其**所有**后代组成。这是演化分析中最自然和最重要的分类单元。在系统发育树上，任何一个内部节点及其所有后代分支和末端节点共同构成一个[支系](@entry_id:171685)。一个[有根树](@entry_id:266860)的**拓扑结构（topology）**，即其分支模式，完全由其所包含的所有[支系](@entry_id:171685)的集合来定义。

理解拓扑结构是一个抽象概念至关重要。我们在纸上或屏幕上看到的[树图](@entry_id:276372)只是该拓扑结构的一种可视化表示。我们可以围绕任何一个内部节点“旋转”其子树，这会改变末端节点的左右顺序，但不会改变树所代表的演化关系。这种**节点旋转（node rotation）**操作不会改变树的拓扑结构，因此也不会改变其所定义的支系集合、简约法得分、似然值或任意两点间的谱系距离。换言之，树的绘制方式是灵活的，但其传达的演化假说——由其拓扑结构定义——是固定的 。

**[并系群](@entry_id:147411)（Paraphyletic Group）**
一个**[并系群](@entry_id:147411)**包含一个共同祖先，但**并非**其所有后代。它是在一个完整的[单系群](@entry_id:142386)中“挖掉”了一个或多个小的[单系群](@entry_id:142386)后形成的。例如，传统分类学中的“爬行类”就是一个典型的[并系群](@entry_id:147411)，因为它包括了爬行动物的共同祖先及其大部分后代（如蜥蜴、蛇、龟、鳄鱼），但排除了由其演化而来的鸟类。

我们可以用[集合论](@entry_id:137783)的语言来形式化地描述这一概念。给定两个[支系](@entry_id:171685) $C_1$ 和 $C_2$，其中 $C_2$ 嵌套在 $C_1$ 内部（$C_2 \subsetneq C_1$），由集合差 $P = C_1 \setminus C_2$ 定义的类群通常是[并系群](@entry_id:147411)。然而，这并非绝对。例如，如果一个节点 $u$ 恰好有两个直接子节点 $w_1$ 和 $w_2$，那么 $C(u) = C(w_1) \cup C(w_2)$。如果我们定义 $C_1 = C(u)$ 和 $C_2 = C(w_1)$，那么集合差 $P = C_1 \setminus C_2 = C(w_2)$。此时，$P$ 本身就是一个完整的[支系](@entry_id:171685)，因而是单系而非并系的。此外，并非所有[并系群](@entry_id:147411)都能通过从一个大[支系](@entry_id:171685)中减去一个小[支系](@entry_id:171685)来表示。例如，要定义“爬行类”，需要从[羊膜动物](@entry_id:167350)（Amniota）这个大支系中同时排除鸟类（Aves）和哺乳类（Mammalia）这两个各自独立的支系，而鸟类和哺乳类的并集本身并不是一个[单系群](@entry_id:142386) 。

**复系群（Polyphyletic Group）**
一个**复系群**由来自两个或多个不同演化谱系的成员组成，但**不包含**它们共同的最近祖先。一个经典的例子是“有翼脊椎动物”，它包括鸟类和蝙蝠。尽管它们都具有翅膀这一相似特征（性状趋同演化的结果），但它们的最近共同祖生（一种早期的、无翼的[羊膜动物](@entry_id:167350)）并不属于这个群体。

形式上，一个群体 $G$ 被定义为复系群，当且仅当该群体所有成员的[最近共同祖先](@entry_id:136722) $\operatorname{MRCA}(G)$ 不在群体 $G$ 之内，即 $\operatorname{MRCA}(G) \notin G$。这个定义的一个必然推论是，该群体的成员必然[分布](@entry_id:182848)在 $\operatorname{MRCA}(G)$ 的至少两个不同的子[支系](@entry_id:171685)中，这反映了其“[多源](@entry_id:170321)”的特性 。

需要强调的是，单系、并系和复系这些概念都依赖于一个确定的根，因为它们都基于“祖先”和“后代”的定义。在[无根树](@entry_id:199885)上，这些概念是无意义的 。

### 定量信息：从[分支图](@entry_id:274587)到[系统发育图](@entry_id:166959)

[系统发育树](@entry_id:140506)不仅能展示[演化关系](@entry_id:175708)，还能承载关于[演化速率](@entry_id:202008)和时间的定量信息，这些信息主要体现在分支的长度上。

**[分支图](@entry_id:274587)（Cladogram）**
最简单的[树图](@entry_id:276372)是**[分支图](@entry_id:274587)**。在[分支图](@entry_id:274587)中，分支的长度没有实际的定量意义，仅用于清晰地展示树的拓扑结构（即分支模式）。我们只关心谁和谁是近亲，而不关心它们之间演化了多少。

**[系统发育图](@entry_id:166959)（Phylogram）**
相比之下，**[系统发育图](@entry_id:166959)**的[分支长度](@entry_id:177486)是经过计算的，其长度与沿该谱系发生的演化变化量成正比。这个变化量通常以“每个位点的期望替换数”为单位来衡量。

从[分支图](@entry_id:274587)到[系统发育图](@entry_id:166959)的转变，是在保持相同拓扑结构和根的前提下，为分支赋予了定量的生物学意义。这个过程增加了[信息量](@entry_id:272315)，但并未改变树的基本结构——节点和支系的定义保持不变 。[分支长度](@entry_id:177486)的生物学解释如下：

*   **末端[分支长度](@entry_id:177486)（Terminal branch length）**：估计了从该末端谱系与其姐妹群（sister group）分化之后，到被采样时为止，所独立积累的演化变化量。这部分变化是该末端类群所特有的。
*   **内部[分支长度](@entry_id:177486)（Internal branch length）**：估计了在一个祖先谱系上，从其与更早的谱系分离后，到它自身分化为两个子谱系之前，所发生的演化变化量。这段分支上发生的遗传变异会被其下游的所有后代成员所继承，是它们共享的演化历史的一部分。

需要强调的是，[系统发育图](@entry_id:166959)中的[分支长度](@entry_id:177486)代表的是演化**变化量**，而不是绝对的**时间**。只有在假设整个树或部分谱系以恒定速率演化（即满足**分子钟(molecular clock)**假说）并进行校准后，[分支长度](@entry_id:177486)才能被转换为时间单位 。

### 在树中导航：路径与距离

一旦分支被赋予了长度，我们就可以在树上定义和计算距离。两个类群之间的**谱系距离（patristic distance）**是在树上连接它们的唯一简单路径（unique simple path）上所有[分支长度](@entry_id:177486)的总和。

这条路径的结构具有明确的几何特性。对于任意两个不同的叶节点 $u$ 和 $v$，连接它们的唯一路径，可以看作是从 $u$ “向上”追溯到它们的[最近共同祖先](@entry_id:136722) $w = \operatorname{MRCA}(u, v)$，然后再“向下”延伸到 $v$ 的过程。这条路径是路径 $P(u, w)$ 和路径 $P(v, w)$ 的并集，并且这两条路径除了共享节点 $w$ 之外，没有其他任何公共节点或分支。

因此，它们之间的谱系距离 $d(u,v)$ 可以精确地表示为它们各自到其[最近共同祖先](@entry_id:136722)的距离之和 ：
$$d(u,v) = d(u,w) + d(v,w)$$
这个简单的加法关系是许多基于距离的系统发育构建方法（如Neighbor-Joining）和演化分析（如计算成对差异）的数学基础。

### 进阶术语：支系定义的细微差别

随着研究的深入，尤其是在结合了化石记录的古生物学中，对“支系”的定义出现了更精细的划分，主要分为基于节点和基于分支的定义。

**基于节点的支系（Node-based Clade）**
这是我们之前讨论的标准定义，也常被称为**冠群（crown group）**。它由一个特定节点（通常是现存物种的[最近共同祖先](@entry_id:136722)）及其所有后代（包括现存和已灭绝的）构成。

**基于分支的支系（Branch-based Clade）**
这种定义也被称为**总群（total group）**，它不是基于一个节点，而是基于亲缘关系。例如，一个总群可以被定义为“所有与物种A的[亲缘关系](@entry_id:172505)比与物种B更近的物种”。这个群体不仅包括以A为代表的冠群，还包括所有在通向该冠群MRCA的“主干”谱系（stem lineage）上分化出去的已灭绝旁支。

这两种定义的区别可以通过一个包含化石的例子来阐明。假设一个演化树中，一个化石物种 $F$ 是一个包含 $A, C, D$ 三个物种的支系的姐妹群。根据基于节点的定义，由 $A, C, D$ 的[最近共同祖先](@entry_id:136722)定义的冠群只包含 $\{A, C, D\}$。然而，如果我们定义一个基于分支的类群，即“所有与 $A$ 的关系比与某个外群 $B$ 更近的物种”，那么这个总群将不仅包含 $\{A, C, D\}$，还包括在通向它们[共同祖先](@entry_id:175919)的谱系上更早分离出去的化石 $F$。因此，这个总群的成员是 $\{A, C, D, F\}$。化石 $F$ 属于总群，但不属于冠群。这个区别对于精确描述包含大量已灭绝类群的演化历史至关重要 。
