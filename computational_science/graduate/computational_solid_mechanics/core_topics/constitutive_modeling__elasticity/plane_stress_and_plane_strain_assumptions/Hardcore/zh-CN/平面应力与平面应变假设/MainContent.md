## 引言
在[固体力学](@entry_id:164042)的广阔领域中，解决完整的三维（3D）问题往往面临巨大的解析和计算挑战。因此，发展精确且高效的简化模型成为工程师与科学家的核心任务之一。[平面应力与平面应变](@entry_id:172357)假设正是其中最基本、应用最广泛的降维工具，它们将复杂的3D应[力场](@entry_id:147325)问题简化为更易于处理的二维（2D）形式，构成了从教科书到高级有限元软件的理论基石。然而，何时选择[平面应力](@entry_id:172193)而非平面应变？这两种理想化模型背后的物理依据是什么？它们在不同学科领域中又有哪些巧妙而深刻的应用？这些问题是初学者常常面临的困惑，也是资深研究者在面对复杂问题时必须审慎思考的要点。

本文旨在系统性地解答这些问题，为读者构建一个关于[平面应力与平面应变](@entry_id:172357)假设的完整知识体系。我们将遵循一条从理论到实践的清晰路径：

-   在“原理与机制”一章中，我们将从三维[弹性理论](@entry_id:184142)出发，严格定义并推导[平面应力与平面应变](@entry_id:172357)的控制方程和本构关系，并通过[渐近分析](@entry_id:160416)阐明其物理合理性。
-   接下来的“应用与跨学科连接”一章将展示这些理论在机械工程、[断裂力学](@entry_id:141480)、[地质力学](@entry_id:175967)乃至多物理场耦合等前沿领域的实际应用，揭示[模型选择](@entry_id:155601)如何影响我们对物理世界的理解。
-   最后，在“动手实践”部分，通过一系列精心设计的思考题，读者将有机会亲手应用所学知识，加深对两种[模型差异](@entry_id:198101)性及其在[计算建模](@entry_id:144775)中重要性的理解。

通过本章的学习，读者将不仅掌握[平面应力与平面应变](@entry_id:172357)的计算方法，更能深刻领会其作为连接理论与工程实践桥梁的精髓。

## 原理与机制

在[连续介质力学](@entry_id:155125)中，对三维（3D）问题进行降维简化是分析复杂结构行为的有力工具。其中，[平面应力与平面应变](@entry_id:172357)假设是工程和地球物理学领域应用最广泛的两种二维（2D）模型。这些模型将完整的3D弹性问题简化为更易于处理的2D形式，极大地降低了分析和计算的复杂性。本章旨在从第一性原理出发，系统地阐述[平面应力与平面应变](@entry_id:172357)假设的定义、物理基础、推导过程及其在特定情况下的适用性与局限性。

### 三维[弹性理论](@entry_id:184142)中的基本定义

我们从一个均匀、各向同性、线性弹性体在小应变假设下的行为出发。其本构关系，即[应力与应变](@entry_id:137374)之间的关系，由胡克定律（Hooke's Law）给出。使用Lamé常数 $\lambda$ 和 $\mu$（剪切模量，也记作 $G$），应力张量 $\boldsymbol{\sigma}$ 和[应变张量](@entry_id:193332) $\boldsymbol{\varepsilon}$ 的关系可以写为：

$ \sigma_{ij} = \lambda \varepsilon_{kk} \delta_{ij} + 2\mu \varepsilon_{ij} $

其中，$\varepsilon_{kk} = \varepsilon_{11} + \varepsilon_{22} + \varepsilon_{33}$ 是应变张量的迹，代表体积应变；$\delta_{ij}$ 是克罗内克符号。Lamé常数与更常见的[工程弹性常数](@entry_id:182223)——杨氏模量 $E$ 和泊松比 $\nu$——相关：

$ \lambda = \frac{E\nu}{(1+\nu)(1-2\nu)}, \quad \mu = G = \frac{E}{2(1+\nu)} $

[平面应力与平面应变](@entry_id:172357)假设本质上是对三维应力或应变场的特定约束，这些约束在不同物理场景下近似成立。

#### 平面应变 (Plane Strain)

**平面应变**是一种运动学约束，它假设物体在某一方向（通常是 $x_3$ 或 $z$ 轴方向）上没有变形。这适用于那些在一个方向上特别长、且其[横截面](@entry_id:154995)上的载荷和几何形状不沿该方向变化的物体，例如长坝、隧道或被刚性端板约束的[棱柱杆](@entry_id:190143)。

该假设的数学定义是，所有与面外方向（$x_3$）相关的应变分量均为零 ：

$ \varepsilon_{33} = 0, \quad \varepsilon_{13} = 0, \quad \varepsilon_{23} = 0 $

这些运动学约束对[位移场](@entry_id:141476)有直接影响。若位移场为 $\mathbf{u} = (u_1, u_2, u_3)$，则 $\varepsilon_{33} = \partial u_3 / \partial x_3 = 0$ 意味着 $u_3$ 不依赖于 $x_3$。同时，$\varepsilon_{13} = \frac{1}{2}(\partial u_1 / \partial x_3 + \partial u_3 / \partial x_1) = 0$ 和 $\varepsilon_{23} = \frac{1}{2}(\partial u_2 / \partial x_3 + \partial u_3 / \partial x_2) = 0$ 结合平面问题的标准假设（即平面内位移不依赖于面外坐标，$u_1=u_1(x_1, x_2)$ 和 $u_2=u_2(x_1, x_2)$），可推导出 $\partial u_3 / \partial x_1 = 0$ 和 $\partial u_3 / \partial x_2 = 0$。综上所述，$u_3$ 必须是一个常数，代表整个物体在 $x_3$ 方向上的刚体位移。

在[平面应变](@entry_id:167046)条件下，面外应力分量会怎样呢？
从本构关系 $\sigma_{13} = 2\mu \varepsilon_{13}$ 和 $\sigma_{23} = 2\mu \varepsilon_{23}$ 可知，面外[剪切应力](@entry_id:137139)也必须为零：

$ \sigma_{13} = 0, \quad \sigma_{23} = 0 $

然而，面外法向应力 $\sigma_{33}$ 通常不为零。为了维持 $\varepsilon_{33} = 0$ 的约束，必须施加一个相应的法向应力。将 $\varepsilon_{33}=0$ 代入[法向应力](@entry_id:260622)的[本构方程](@entry_id:138559)：

$ \sigma_{33} = \lambda(\varepsilon_{11} + \varepsilon_{22} + 0) + 2\mu(0) = \lambda(\varepsilon_{11} + \varepsilon_{22}) $

这个非零的 $\sigma_{33}$ 是阻止物体在 $x_3$ 方向发生[泊松效应](@entry_id:158876)变形所必需的“约束应力” 。

#### 平面应力 (Plane Stress)

**平面应力**是一种应力状态的假设，它假设物体在某一方向（$x_3$）上是“薄”的，以至于所有作用在该薄平面上的应力分量可以忽略不计。这适用于薄板、薄壳和[膜结构](@entry_id:183960)，其载荷主要作用于板平面内。

该假设的数学定义是，所有与面外方向（$x_3$）相关的应力分量均为零 ：

$ \sigma_{33} = 0, \quad \sigma_{13} = 0, \quad \sigma_{23} = 0 $

这个假设直接要求在物体的两个自由表面（例如 $x_3 = \pm h/2$）上没有施加法向和切向的牵[引力](@entry_id:175476)。正如我们将看到的，这是此假设成立的必要条件 。

在[平面应力条件](@entry_id:168184)下，面外应变分量又会怎样呢？
与[平面应变](@entry_id:167046)情况类似，$\sigma_{13} = 2\mu \varepsilon_{13} = 0$ 和 $\sigma_{23} = 2\mu \varepsilon_{23} = 0$ 意味着面外剪切应变也必须为零：

$ \varepsilon_{13} = 0, \quad \varepsilon_{23} = 0 $

然而，面外[法向应变](@entry_id:204633) $\varepsilon_{33}$ 通常不为零。由于平面[内应力](@entry_id:193721) $\sigma_{11}$ 和 $\sigma_{22}$ 的存在，物体会通过[泊松效应](@entry_id:158876)在厚度方向上发生伸长或收缩。我们可以从[本构关系](@entry_id:186508)中求出 $\varepsilon_{33}$。从 $\sigma_{33}$ 的方程入手：

$ \sigma_{33} = \lambda(\varepsilon_{11} + \varepsilon_{22} + \varepsilon_{33}) + 2\mu \varepsilon_{33} = 0 $

整理后可得：

$ (\lambda + 2\mu)\varepsilon_{33} = -\lambda(\varepsilon_{11} + \varepsilon_{22}) $

解出 $\varepsilon_{33}$，并用 $E$ 和 $\nu$ 表示系数：

$ \varepsilon_{33} = -\frac{\lambda}{\lambda+2\mu}(\varepsilon_{11} + \varepsilon_{22}) = -\frac{\nu}{1-\nu}(\varepsilon_{11} + \varepsilon_{22}) $

这个关系式精确地量化了[平面应力](@entry_id:172193)状态下的厚度变化，这是它与[平面应变](@entry_id:167046)最核心的区别之一 。在平面应力中，$\sigma_{33}=0$ 但 $\varepsilon_{33}\neq 0$；而在[平面应变](@entry_id:167046)中，$\varepsilon_{33}=0$ 但 $\sigma_{33}\neq 0$。

### 二维控制方程的简化

一旦采纳了平面应力或[平面应变假设](@entry_id:186003)，完整的三维[平衡方程](@entry_id:172166)和[本构关系](@entry_id:186508)就可以被简化为一套自洽的二维[方程组](@entry_id:193238)，这对于解析求解和数值模拟（如有限元法）至关重要。

三维静态[平衡方程](@entry_id:172166)（无体力时）为 $\sigma_{ij,j} = 0$。在[笛卡尔坐标系](@entry_id:169789)下，前两个分量为：
$ \frac{\partial \sigma_{11}}{\partial x_1} + \frac{\partial \sigma_{12}}{\partial x_2} + \frac{\partial \sigma_{13}}{\partial x_3} = 0 $
$ \frac{\partial \sigma_{21}}{\partial x_1} + \frac{\partial \sigma_{22}}{\partial x_2} + \frac{\partial \sigma_{23}}{\partial x_3} = 0 $

在**[平面应力](@entry_id:172193)**中，我们假设 $\sigma_{13}=\sigma_{23}=0$ 处处成立。
在**[平面应变](@entry_id:167046)**中，我们假设所有场量不依赖于 $x_3$ 坐标，即 $\partial/\partial x_3 = 0$。同时，由于 $\varepsilon_{13}=\varepsilon_{23}=0$, 可知 $\sigma_{13}=\sigma_{23}=0$。

在这两种情况下，上述三维[平衡方程](@entry_id:172166)中包含面外分量的项都消失了。如果考虑体力 $b_i$，并假设体力也仅作用于平面内（$b_3=0$），则两种情况下的平面内[平衡方程](@entry_id:172166)形式完全相同 ：

$ \frac{\partial \sigma_{11}}{\partial x_1} + \frac{\partial \sigma_{12}}{\partial x_2} + b_1 = 0 $
$ \frac{\partial \sigma_{21}}{\partial x_1} + \frac{\partial \sigma_{22}}{\partial x_2} + b_2 = 0 $

尽管[平衡方程](@entry_id:172166)形式相同，但两种情况下的有效二维本构关系是不同的，这反映了面外约束的差异。通过代数运算，可以从三维[本构关系](@entry_id:186508)中消去面外分量，得到仅包含平面内应力 $\{ \sigma_{11}, \sigma_{22}, \sigma_{12} \}^T$ 和平面内应变 $\{ \varepsilon_{11}, \varepsilon_{22}, \gamma_{12} \}^T$ (其中 $\gamma_{12}=2\varepsilon_{12}$) 的关系式，形如 $\{ \sigma_{in} \} = [Q] \{ \varepsilon_{in} \}$。

对于**平面应力**，有效[本构矩阵](@entry_id:164908) $[Q^{ps}]$ 为：
$$
[Q^{ps}] = \frac{E}{1-\nu^2}
\begin{pmatrix}
1  & \nu  & 0 \\
\nu  & 1  & 0 \\
0  & 0  & \frac{1-\nu}{2}
\end{pmatrix}
$$

对于**[平面应变](@entry_id:167046)**，有效[本构矩阵](@entry_id:164908) $[Q^{pe}]$ 为：
$$
[Q^{pe}] = \frac{E}{(1+\nu)(1-2\nu)}
\begin{pmatrix}
1-\nu  & \nu  & 0 \\
\nu  & 1-\nu  & 0 \\
0  & 0  & \frac{1-2\nu}{2}
\end{pmatrix}
$$
这些矩阵 $[Q]$ 是有限元软件中用于二维单元刚度计算的核心。可以看出，尽管[平衡方程](@entry_id:172166)形式一致，但由于材料[响应矩阵](@entry_id:754302)不同，[平面应力与平面应变](@entry_id:172357)问题会得到截然不同的解 。

### 通过[渐近分析](@entry_id:160416)和[圣维南原理](@entry_id:165302)的物理解释

到目前为止，我们仅在数学上定义了两种理想化模型。然而，在何种物理条件下这些模型是真实三维行为的良好近似？答案蕴含在[渐近分析](@entry_id:160416)和[圣维南原理](@entry_id:165302)（Saint-Venant's Principle）中。

#### 平面应力的物理合理性：薄板近似

考虑一个厚度为 $h$、[特征面](@entry_id:747281)内尺寸为 $L$ 的薄板（$h \ll L$）。其上下表面 $x_3 = \pm h/2$ 是自由的（无牵[引力](@entry_id:175476)），且仅受到不沿厚度变化的平面[内载荷](@entry_id:195654)。我们可以通过量级分析来考察面外应力的大小 。

[自由表面边界条件](@entry_id:749579)意味着 $\sigma_{i3}(x_1, x_2, \pm h/2) = 0$。从第三个平衡方程 $\sigma_{3j,j} = 0$ (假设体力为零) 出发，积分可得：
$ \sigma_{33}(x_1, x_2, x_3) = -\int_{-h/2}^{x_3} (\frac{\partial \sigma_{13}}{\partial x_1} + \frac{\partial \sigma_{23}}{\partial x_2}) dx_3' $

类似地，从前两个平衡方程积分可得 $\sigma_{\alpha 3}$ ($\alpha=1,2$) 的量级。假设平面内应力 $\sigma_{\alpha\beta}$ 的特征尺度为 $\Sigma$，其变化的空间尺度为 $L$。那么，对第一个[平衡方程](@entry_id:172166)沿厚度积分可以估计出 $\sigma_{13}$ 的量级。结果表明：
- 横向[剪切应力](@entry_id:137139) $\sigma_{\alpha 3}$ 的量级为 $O(h/L) \cdot \Sigma$。
- 横向法向应力 $\sigma_{33}$ 的量级为 $O((h/L)^2) \cdot \Sigma$。

由于 $h/L \ll 1$，面外剪切应力相对于平面[内应力](@entry_id:193721)是一个小量，而面外法向应力则是一个更高阶的小量。因此，在薄板的“内部”区域，忽略所有面外应力分量（即采用[平面应力假设](@entry_id:184389)）是一个渐近一致的近似。

需要强调的是，这种近似在靠近载荷施加处或几何突变处的“[边界层](@entry_id:139416)”区域内可能失效。根据[圣维南原理](@entry_id:165302)，这些三维效应的[影响范围](@entry_id:166501)与板厚 $h$ 相当。在远离这些区域的地方，应[力场](@entry_id:147325)会迅速衰减到平面应力状态 。

#### 平面应变的物理合理性：长棱柱体近似

考虑一个在 $x_3$ 方向很长（$L \gg$ [横截面](@entry_id:154995)尺寸）的棱柱体，其两端被固定以防止轴向位移（例如 $u_3(x_1, x_2, 0) = u_3(x_1, x_2, L) = 0$）。横向载荷沿杆长不变。根据[圣维南原理](@entry_id:165302)，在远离端部的“内部”区域，[应力与应变](@entry_id:137374)场应趋于与轴向坐标 $x_3$ 无关。

一个与 $x_3$ 无关的解，如果处于[平面应力](@entry_id:172193)状态（$\sigma_{33}=0$），则必然导致非零的[轴向应变](@entry_id:160811) $\varepsilon_{33}$，进而产生轴向位移 $u_3$ 正比于 $x_3$。这与端部固定的边界条件相矛盾。

相反，一个[平面应变](@entry_id:167046)状态（$\varepsilon_{33}=0, \varepsilon_{13}=0, \varepsilon_{23}=0$）意味着 $u_3$ 必须为常数。结合端部约束 $u_3=0$，可得内部区域的 $u_3$ 处处为零。这与物理情景完全吻合。因此，在远离端部的内部，系统自然趋向于[平面应变](@entry_id:167046)状态 。由端部约束引起的任何三维扰动场，会以指数形式沿轴向衰减，其[特征衰减长度](@entry_id:183295)与[横截面](@entry_id:154995)的最大特征尺寸（例如矩形[截面](@entry_id:154995)的 $\max\{b,h\}$）成正比。

### 高等专题与特殊情况

[平面应力与平面应变](@entry_id:172357)的概念可以进一步扩展和深化，以处理更复杂的情况。

#### [广义平面应变](@entry_id:182960) (Generalized Plane Strain)

在[平面应变](@entry_id:167046)中，我们强制 $\varepsilon_{33}=0$。在[平面应力](@entry_id:172193)中，$\varepsilon_{33}$ 通常是一个随 $x_1, x_2$ 变化的量，但在多数简化模型中，为保证[运动学](@entry_id:173318)协调性，它被假定为常数。**[广义平面应变](@entry_id:182960)**则介于两者之间，它允许面外[法向应变](@entry_id:204633)是一个非零的常数，即 $\varepsilon_{33} = \varepsilon_0$ 。

这个假设对应的[位移场](@entry_id:141476)为 $u_3(x_1, x_2, x_3) = \varepsilon_0 z + c$，其中 $\varepsilon_0$ 和 $c$ 是常数（同时保持 $\varepsilon_{13}=\varepsilon_{23}=0$）。[广义平面应变](@entry_id:182960)模型非常适用于分析受到均匀轴向拉伸或压缩，或者受到[纯弯曲](@entry_id:202969)作用的长棱柱体。它比标准平面应变模型更具灵活性。

#### 不可压缩极限 ($\nu \to 0.5$)

材料的泊松比 $\nu$ 趋于 $0.5$ 时，材料表现出[不可压缩性](@entry_id:274914)。在此极限下，[平面应力与平面应变](@entry_id:172357)模型的响应截然不同，这揭示了它们在约束上的深刻差异。

我们可以定义一个二维有效[体积模量](@entry_id:160069) $\kappa_{2D}$，它描述了平面内平均正应力 $p_{2D} = \frac{1}{2}(\sigma_{11}+\sigma_{22})$ 与平面内面积应变 $e = \varepsilon_{11}+\varepsilon_{22}$ 之间的关系。

- 在**平面应变**下，$\kappa_{2D}^{(\text{pe})} = \lambda + \mu$。当 $\nu \to 0.5$ 时，$\lambda \to \infty$，因此 $\kappa_{2D}^{(\text{pe})} \to \infty$。这意味着在[平面应变](@entry_id:167046)条件下，对[不可压缩材料](@entry_id:159741)施加面积应变需要无穷大的应力。这种现象被称为**[体积锁定](@entry_id:172606) (volumetric locking)**，是有限元分析中需要特别处理的数值难题。
- 在**平面应力**下，$\kappa_{2D}^{(\text{ps})}$ 在 $\nu \to 0.5$ 极限下趋于一个有限值 $3\mu$。这是因为材料可以通过厚度方向的自由变形（$\varepsilon_{33} \neq 0$）来保持体积不变，从而避免了平面内的刚度剧增。

这个例子生动地说明，[运动学](@entry_id:173318)约束（[平面应变](@entry_id:167046)）在不可压缩极限下会导致刚度伪影，而[应力约束](@entry_id:201787)（[平面应力](@entry_id:172193)）则不会 。

#### [各向异性材料](@entry_id:184874)与极限的不可交换性

对于各向异性材料，例如[纤维增强复合材料](@entry_id:194995)，[平面应力与平面应变](@entry_id:172357)模型的差异更加微妙且重要。考虑一个横观[各向同性材料](@entry_id:170678)，其对称轴为 $x_3$ 方向。这种情况常见于[单向复合材料](@entry_id:196178)层合板。

我们可以考虑两种不同的极限过程 ：
1.  **薄层极限 (R1)**：先应用[平面应力假设](@entry_id:184389)（$\sigma_{33}=0$）得到有效二维刚度，然后考虑材料在厚度方向非常硬（$C_{33} \to \infty$）的情况。这代表了单个薄[复合材料](@entry_id:139856)层（lamina）的行为。
2.  **厚板极限 (R2)**：先考虑材料在厚度方向非常硬（$C_{33} \to \infty$），这导致[运动学](@entry_id:173318)上 $\varepsilon_{33} \to 0$，然后再应用[平面应变假设](@entry_id:186003)。这代表了厚层合板（laminate）或受约束的[复合材料](@entry_id:139856)结构的行为。

通过推导可以证明，对于面内-面外耦合刚度 $C_{13} \neq 0$ 的材料，这两种极限过程得到的结果是不同的，即 $[Q^{ps}]$ 和 $[Q^{pe}]$ 的极限不相同。这揭示了一个深刻的建模原则：物理极限（薄板 vs 厚板）与数学简化（[平面应力](@entry_id:172193) vs [平面应变](@entry_id:167046)）的顺序不可交换。

这个结论的实践意义是：对于薄的[复合材料](@entry_id:139856)结构，必须使用[平面应力](@entry_id:172193)模型；而对于厚的、或受到面外约束的[复合材料](@entry_id:139856)结构，[平面应变](@entry_id:167046)模型（或完整的三维模型）才是恰当的选择。混用模型会导致对材料平面内刚度的严重错误估计，从而影响[结构分析](@entry_id:153861)的准确性。
