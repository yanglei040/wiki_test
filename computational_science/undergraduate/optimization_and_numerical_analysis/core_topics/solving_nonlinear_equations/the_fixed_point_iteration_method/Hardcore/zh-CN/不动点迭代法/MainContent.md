## 引言
[不动点迭代法](@entry_id:168837)是[数值分析](@entry_id:142637)乃至整个计算科学领域中一块至关重要的基石。它不仅为求解形式复杂的方程提供了一种强大而优雅的通用框架，更是一种深刻的数学思想，能够将不同学科中看似无关的“平衡”与“求解”问题联系起来。从预测行星轨道到优化机器学习模型，[不动点](@entry_id:156394)的概念无处不在，而掌握其迭代求解方法，则是理解这些高级应用计算本质的关键。

然而，如何将一个问题有效地转化为收敛的[不动点迭代](@entry_id:749443)？我们如何判断一个迭代过程是[稳定收敛](@entry_id:199422)、[振荡](@entry_id:267781)逼近还是迅速发散？又如何设计出[收敛速度](@entry_id:636873)更快的算法？本文旨在系统性地回答这些问题，为读者构建一个关于[不动点迭代法](@entry_id:168837)的完整知识体系。

本文将分为三个核心章节。在第一章**“原则与机制”**中，我们将深入[不动点迭代法](@entry_id:168837)的数学心脏，详细剖析其基本概念、[收敛性与稳定性](@entry_id:636533)判据、收敛速率与阶，并介绍保证收敛性的核心定理。接着，在第二章**“应用与跨学科联系”**中，我们将视野拓宽，探索该方法在[求解方程组](@entry_id:152624)、分析动态系统、博弈论以及现代数据科学等众多领域的具体应用，领略其惊人的普适性。最后，在第三章**“动手实践”**中，你将通过解决一系列精心设计的问题，亲手实践所学理论，将抽象的知识转化为解决实际问题的能力。现在，让我们从最基本的原则开始，踏上这段探索之旅。

## 原则与机制

### [不动点迭代法](@entry_id:168837)的基本概念

在[数值分析](@entry_id:142637)中，许多问题的核心是求解形如 $f(x) = 0$ 的方程。一种强大而通用的策略是将该方程代数地重构为一个等价的**[不动点](@entry_id:156394)问题**（fixed-point problem）。如果一个点 $x^*$ 满足方程 $x^* = g(x^*)$，那么我们称 $x^*$ 是函数 $g(x)$ 的一个**[不动点](@entry_id:156394)**（fixed point）。从图形上看，函数 $y=g(x)$ 的[不动点](@entry_id:156394)正是其图像与直线 $y=x$ 的交点。

将[求根问题](@entry_id:174994) $f(x)=0$ 转化为[不动点](@entry_id:156394)问题 $x=g(x)$ 的方法并不唯一。例如，为了求解 $f(x) = x^3 - x - 1 = 0$，我们可以构造出多种等价的[不动点](@entry_id:156394)形式，如 $x = x^3 - 1$ 或 $x = (x+1)^{1/3}$ 或 $x = x - \frac{x^3-x-1}{3x^2-1}$（这正是牛顿法，我们稍后会详细讨论）。选择一个“好”的迭代函数 $g(x)$ 对于算法的成功至关重要。

一旦我们确定了[不动点](@entry_id:156394)形式 $x=g(x)$，就可以采用**[不动点迭代法](@entry_id:168837)**（fixed-point iteration method）来逼近解。该方法从一个初始猜测值 $x_0$ 开始，通过以下递推关系生成一个序列 $\{x_n\}_{n=0}^{\infty}$：

$$x_{n+1} = g(x_n)$$

如果这个序列收敛于某个值 $x^*$，即 $\lim_{n\to\infty} x_n = x^*$，并且函数 $g(x)$ 是连续的，那么我们可以对上式两边取极限，得到 $x^* = g(x^*)$。这意味着[序列的极限](@entry_id:159239)就是 $g(x)$ 的一个[不动点](@entry_id:156394)。

让我们通过一个具体模型来理解这个过程。考虑一个化学反应器中的某个过程 。假设某种活性成分的初始浓度为 $C_0$。在每个周期中，浓度会先因反应而减少其当前值的 $\alpha$ 倍，然后再通过添加固定量物质使其增加一个常数 $\beta$。设 $C_n$ 为第 $n$ 个周期结束后的浓度，其动态变化过程可以用以下[线性差分方程](@entry_id:178777)描述：

$$C_{n+1} = C_n - \alpha C_n + \beta = (1-\alpha)C_n + \beta$$

这个方程正是一个[不动点迭代](@entry_id:749443)的形式，其中迭代函数为 $g(C) = (1-\alpha)C + \beta$。如果这个过程达到一个稳定的平衡状态，即浓度不再变化，我们称之为**[稳态](@entry_id:182458)浓度**（steady-state concentration），记作 $C^*$。此时，应有 $C^* = g(C^*)$，即：

$$C^* = (1-\alpha)C^* + \beta$$

通过简单的代数运算，我们可以解出这个[不动点](@entry_id:156394)：

$$\alpha C^* = \beta \quad \implies \quad C^* = \frac{\beta}{\alpha}$$

这个 $C^*$ 就是迭代序列期望收敛到的目标。通过求解该[线性递推关系](@entry_id:273376)，可以得到浓度 $C_n$ 的显式表达式：

$$C_n = \left(C_0 - \frac{\beta}{\alpha}\right)(1-\alpha)^n + \frac{\beta}{\alpha}$$

从这个表达式中可以清晰地看到收敛的条件。当且仅当 $|1-\alpha|  1$ (即 $0  \alpha  2$)时，随着 $n \to \infty$，$(1-\alpha)^n$ 项趋于零，使得 $\lim_{n\to\infty} C_n = \frac{\beta}{\alpha}$。这个例子直观地展示了[不动点](@entry_id:156394)、迭代过程以及[收敛条件](@entry_id:166121)之间的联系。

### [不动点](@entry_id:156394)的[收敛性与稳定性](@entry_id:636533)

一个自然而然的问题是：对于任意给定的迭代函数 $g(x)$ 和初始值 $x_0$，迭代序列 $\{x_n\}$ 是否总能收敛到一个[不动点](@entry_id:156394)？答案是否定的。收敛性强烈地依赖于函数 $g(x)$ 在[不动点](@entry_id:156394)附近的局部行为，特别是其[导数的性质](@entry_id:141529)。

我们可以通过几何直观来理解这一点。在[坐标系](@entry_id:156346)中，从点 $(x_n, x_n)$ 出发，垂直移动到曲线 $y=g(x)$ 上的点 $(x_n, g(x_n)) = (x_n, x_{n+1})$，然后再水平移动到直线 $y=x$ 上的点 $(x_{n+1}, x_{n+1})$，就完成了一次迭代。重复此过程，可以绘制出所谓的“[阶梯图](@entry_id:263049)”或“蛛网图”，直观地展示序列的走向。

**局部收敛性定理**为我们提供了判断收敛性的关键工具。该定理指出：假设 $x^*$ 是 $g(x)$ 的一个[不动点](@entry_id:156394)，且 $g(x)$ 在包含 $x^*$ 的某个开区间上是连续可微的。

-   如果 $|g'(x^*)|  1$，那么存在 $x^*$ 的一个邻域，对于任何初始值 $x_0$ 在该邻域内，[不动点迭代](@entry_id:749443)序列 $x_{n+1} = g(x_n)$ 均收敛到 $x^*$。
-   如果 $|g'(x^*)| > 1$，那么对于任何不等于 $x^*$ 的初始值 $x_0$，迭代序列都不会收敛到 $x^*$。事实上，只要迭代值进入 $x^*$ 的某个特定邻域，它就会被“推离”$x^*$。

基于这个定理，我们可以将[不动点](@entry_id:156394)进行分类：
-   **[吸引不动点](@entry_id:181694)**（Attracting fixed point）：如果 $|g'(x^*)|  1$。
-   **[排斥不动点](@entry_id:189650)**（Repelling fixed point）：如果 $|g'(x^*)| > 1$。
-   **中性[不动点](@entry_id:156394)**（Neutral fixed point）：如果 $|g'(x^*)| = 1$。在这种情况下，仅靠[一阶导数](@entry_id:749425)无法判断其稳定性，需要更高阶的分析。

为什么导数的大小如此关键？设 $e_n = x_n - x^*$ 为第 $n$ 步的误差。根据中值定理，在 $x_n$ 和 $x^*$ 之间存在一个点 $\xi_n$，使得：

$$x_{n+1} - x^* = g(x_n) - g(x^*) = g'(\xi_n)(x_n - x^*)$$

因此，误差的传递关系为 $e_{n+1} = g'(\xi_n) e_n$。当 $x_n$ 足够接近 $x^*$ 时，$\xi_n$ 也接近 $x^*$，于是我们有近似关系：

$$|e_{n+1}| \approx |g'(x^*)| |e_n|$$

如果 $|g'(x^*)|  1$，误差在每次迭代中都会被压缩，趋向于零。相反，如果 $|g'(x^*)| > 1$，误差则会被放大，导致迭代发散 。

让我们看一个具体的例子 。考虑迭代函数 $g(x) = \frac{2x}{1+x^2}$。为了找到其[不动点](@entry_id:156394)，我们求解 $x = g(x)$：

$$x = \frac{2x}{1+x^2} \implies x(1+x^2) = 2x \implies x^3 - x = 0 \implies x(x-1)(x+1) = 0$$

因此，该函数有三个[不动点](@entry_id:156394)：$x^*_1 = -1$, $x^*_2 = 0$, $x^*_3 = 1$。为了分析它们的稳定性，我们计算其导数：

$$g'(x) = \frac{2(1+x^2) - 2x(2x)}{(1+x^2)^2} = \frac{2(1-x^2)}{(1+x^2)^2}$$

现在，在每个[不动点](@entry_id:156394)处评估导数的大小：
-   对于 $x^*=0$：$|g'(0)| = |\frac{2(1-0)}{(1+0)^2}| = 2 > 1$。因此，$x=0$ 是一个**[排斥不动点](@entry_id:189650)**。
-   对于 $x^*=1$：$|g'(1)| = |\frac{2(1-1)}{(1+1)^2}| = 0  1$。因此，$x=1$ 是一个**[吸引不动点](@entry_id:181694)**。
-   对于 $x^*=-1$：$|g'(-1)| = |\frac{2(1-1)}{(1+1)^2}| = 0  1$。因此，$x=-1$ 也是一个**[吸引不动点](@entry_id:181694)**。

这个例子清晰地表明，同一个函数可以同时拥有不同稳定性的[不动点](@entry_id:156394)。

### 收敛的性态与速率

当迭代收敛时，其逼近[不动点](@entry_id:156394)的方式（性态）和速度（速率）同样由导数 $g'(x^*)$ 决定。

#### 单调收敛与[振荡](@entry_id:267781)收敛

导数的**符号**决定了收敛的性态：
-   如果 $0 \le g'(x^*)  1$，那么在 $x^*$ 附近，$e_{n+1} \approx g'(x^*) e_n$ 表明 $e_{n+1}$ 和 $e_n$ 的符号相同。这意味着如果 $x_n$ 在 $x^*$ 的一侧，那么 $x_{n+1}$ 也在同一侧。迭代序列将从一侧单调地逼近[不动点](@entry_id:156394)，形成“阶梯状”的收敛路径。
-   如果 $-1  g'(x^*)  0$，那么 $e_{n+1}$ 和 $e_n$ 的符号相反。这意味着如果 $x_n$ 大于 $x^*$，那么 $x_{n+1}$ 将小于 $x^*$，反之亦然。迭代序列将在[不动点](@entry_id:156394)两侧来回[振荡](@entry_id:267781)，逐步逼近[不动点](@entry_id:156394)，形成“蛛网状”或“螺旋状”的收敛路径。

一个经典的例子是求解方程 $x = \cos(x)$ 。该方程的唯一解被称为**Dottie数**。我们可以使用迭代 $x_{n+1} = \cos(x_n)$ 来求解。设[不动点](@entry_id:156394)为 $x^*$，迭代函数为 $g(x) = \cos(x)$，其导数为 $g'(x) = -\sin(x)$。由于[不动点](@entry_id:156394) $x^*$ 位于区间 $(0, 1)$（[弧度](@entry_id:171693)），在该区间内 $\sin(x^*) > 0$。因此，$g'(x^*) = -\sin(x^*)$ 是一个负数。同时，由于 $\sin(x^*) = \sqrt{1 - \cos^2(x^*)} = \sqrt{1 - (x^*)^2}  1$，我们有 $-1  g'(x^*)  0$。这表明，对Dottie数的迭代收敛是一种[振荡](@entry_id:267781)收敛。

#### 收敛速率

从近似关系式 $|e_{n+1}| \approx |g'(x^*)| |e_n|$ 可以看出，误差在每一步中大约乘以一个常数因子 $|g'(x^*)|$。这种收敛行为被称为**[线性收敛](@entry_id:163614)**（linear convergence），而常数 $|g'(x^*)|$ 被称为**渐进[误差常数](@entry_id:168754)**（asymptotic error constant）。这个值越小，收敛得越快。

这个关系式不仅是理论上的，它还有重要的实践意义。通过递推，我们有 $|e_N| \approx |g'(x^*)|^N |e_0|$。假设我们知道一个收缩常数 $L$（即 $|g'(x)| \le L  1$），那么我们可以得到一个严格的[误差界](@entry_id:139888)：

$$|x_N - x^*| \le L^N |x_0 - x^*|$$

这个不等式使我们能够预估达到特定精度所需的迭代次数。例如，假设在一个[数值模拟](@entry_id:137087)中，已知迭代函数的导数满足 $|g'(c)| \le 0.4$ 。如果我们希望将初始误差 $|c_0 - c^*|$ 减小至少 $10^5$ 倍，即要求 $|c_N - c^*| \le 10^{-5} |c_0 - c^*|$，我们可以通过求解以下不等式来确定所需的最小迭代次数 $N$：

$$0.4^N \le 10^{-5}$$

对两边取对数，我们得到：

$$N \ln(0.4) \le -5 \ln(10) \quad \implies \quad N \ge \frac{-5 \ln(10)}{\ln(0.4)} \approx 12.56$$

因此，我们需要至少 $N=13$ 次迭代来保证达到所需的精度。

### 收敛性保证：[不动点定理](@entry_id:143811)

局部收敛性定理告诉我们，只要初始猜测足够接近[不动点](@entry_id:156394)，迭代就会收敛。但在实际应用中，我们往往希望得到更强的保证：只要初始值在某个指定的区间内，迭代就**一定**会收敛。**[不动点定理](@entry_id:143811)**（Fixed-Point Theorem），或更准确地说是实数线上的**[巴拿赫不动点定理](@entry_id:146620)**（Banach Fixed-Point Theorem），提供了这样的保证。

该定理指出，对于一个定义在闭区间 $I = [a, b]$ 上的函数 $g(x)$，如果满足以下两个条件：
1.  **区间自映射**（Invariant Interval）：函数 $g(x)$ 将区间 $I$ 映射到其自身。也就是说，对于任意 $x \in I$，都有 $g(x) \in I$。
2.  **[压缩映射](@entry_id:139989)**（Contraction Mapping）：存在一个常数 $k$ 满足 $0 \le k  1$，使得对于区间 $I$ 中的任意两点 $x$ 和 $y$，都有 $|g(x) - g(y)| \le k|x - y|$。

如果 $g(x)$ 在 $I$ 上可微，那么第二个条件（压缩映射）可以通过一个更易于验证的充分条件来保证：$|g'(x)| \le k  1$ 对所有 $x \in I$ 成立。

如果这两个条件都得到满足，那么定理保证：
-   在区间 $I$ 中存在**唯一**的[不动点](@entry_id:156394) $x^*$。
-   对于**任何**初始值 $x_0 \in I$，[不动点迭代](@entry_id:749443)序列 $x_{n+1} = g(x_n)$ 都将收敛到 $x^*$。

让我们看如何验证这些条件。考虑函数 $g(x) = 1 + \frac{1}{1+x}$ 和区间 $I=[1, 2]$ 。首先检查区间自映射。函数 $g(x)$ 的导数为 $g'(x) = -\frac{1}{(1+x)^2}$，在区间 $[1, 2]$ 上恒为负，所以 $g(x)$ 是单调递减的。因此，其在 $[1, 2]$ 上的值域为 $[g(2), g(1)] = [1+\frac{1}{3}, 1+\frac{1}{2}] = [\frac{4}{3}, \frac{3}{2}]$。由于 $[\frac{4}{3}, \frac{3}{2}] \subset [1, 2]$，区间自映射条件成立。

另一个更完整的例子是求解热敏电阻的平衡温度 $T^4 - T = 3$，其[不动点](@entry_id:156394)形式为 $T = (T+3)^{1/4}$，区间为 $I=[1, 2]$ 。
-   **验证自映射**：对于 $T \in [1, 2]$，我们有 $4 \le T+3 \le 5$。由于 $T \mapsto T^{1/4}$ 是增函数，所以 $g(T)$ 的值域在 $[4^{1/4}, 5^{1/4}]$ 之内。因为 $1  4^{1/4}$ 且 $5^{1/4}  16^{1/4} = 2$，所以 $g(I) \subset (1, 2) \subset [1, 2]$。自映射条件满足。
-   **验证压缩映射**：导数为 $g'(T) = \frac{1}{4}(T+3)^{-3/4}$。在区间 $[1, 2]$ 上，$g'(T)$ 是正的且单调递减。其最大值在 $T=1$ 处取得：$g'(1) = \frac{1}{4}(4)^{-3/4} = \frac{1}{4 \cdot 2^{3/2}} = \frac{1}{8\sqrt{2}}  1$。因此，我们可以取 $k = \frac{1}{8\sqrt{2}}$，使得对所有 $T \in [1, 2]$ 都有 $|g'(T)| \le k  1$。压缩条件满足。

由于两个条件都成立，[不动点定理](@entry_id:143811)保证迭代 $T_{n+1} = (T_n+3)^{1/4}$ 对于任何从 $[1, 2]$ 出发的初始值，都将收敛到方程在该区间内的唯一解。

### 收敛阶与方法设计

[线性收敛](@entry_id:163614)在很多情况下已经足够，但我们能否设计出收敛更快的迭代方法？答案是肯定的。收敛的“阶”是衡量其速度的一个更精确的指标。

如果一个序列 $x_n$ 收敛到 $x^*$，且存在常数 $\lambda > 0$ 和 $\alpha \ge 1$ 使得：
$$\lim_{n\to\infty} \frac{|x_{n+1} - x^*|}{|x_n - x^*|^\alpha} = \lambda$$
则称该序列的**收敛阶**（order of convergence）为 $\alpha$。
-   $\alpha=1$ 对应[线性收敛](@entry_id:163614)（要求 $\lambda  1$）。
-   $\alpha=2$ 对应**二次收敛**（quadratic convergence）。
-   $\alpha=3$ 对应**[三次收敛](@entry_id:168106)**（cubic convergence）。

[收敛阶](@entry_id:146394)越高，意味着当迭代值接近[不动点](@entry_id:156394)时，每一步迭代能获得的精确[有效位数](@entry_id:190977)越多，收敛速度也越快。

[收敛阶](@entry_id:146394)与迭代函数在[不动点](@entry_id:156394)处的导数密切相关。通过对 $g(x)$ 在 $x^*$ 处进行[泰勒展开](@entry_id:145057)，我们有：
$$x_{n+1} = g(x_n) = g(x^*) + g'(x^*)(x_n-x^*) + \frac{g''(x^*)}{2!}(x_n-x^*)^2 + \dots$$
代入 $e_n = x_n - x^*$ 和 $x^*=g(x^*)$，得到误差传递方程：
$$e_{n+1} = g'(x^*) e_n + \frac{g''(x^*)}{2} e_n^2 + O(e_n^3)$$

-   如果 $g'(x^*) \neq 0$，则误差主要是线性项 $g'(x^*)e_n$ 主导，收敛阶为1（[线性收敛](@entry_id:163614)）。
-   如果 $g'(x^*) = 0$ 但 $g''(x^*) \neq 0$，则误差主要由二次项主导，$|e_{n+1}| \approx |\frac{g''(x^*)}{2}| |e_n|^2$。这正是二次收敛。

大名鼎鼎的**[牛顿法](@entry_id:140116)**（Newton's method）就是二次收敛的一个绝佳例子 。为了求解 $f(x)=0$，牛顿法的迭代公式为 $x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$。这可以看作是迭代函数为 $g(x) = x - \frac{f(x)}{f'(x)}$ 的[不动点迭代](@entry_id:749443)。该迭代函数的导数为：
$$g'(x) = 1 - \frac{f'(x)f'(x) - f(x)f''(x)}{[f'(x)]^2} = \frac{f(x)f''(x)}{[f'(x)]^2}$$
在根 $r$ 处（即 $f(r)=0$），我们有 $g'(r) = 0$（只要 $f'(r) \neq 0$）。因此，牛顿法至少是二次收敛的。与简单的代数重排（如求解 $x^3-x-1=0$ 时的 $x_{k+1}=(x_k+1)^{1/3}$）相比，牛顿法的收敛速度要快得多，因为后者的 $g'(r) = \frac{1}{3r^2} \neq 0$，仅为[线性收敛](@entry_id:163614)。

我们可以进一步设计更高阶的收敛方法。一个迭代方法具有 $m$ 阶收敛，当且仅当在[不动点](@entry_id:156394) $p$ 处满足：
$$g'(p) = g''(p) = \dots = g^{(m-1)}(p) = 0, \quad \text{且} \quad g^{(m)}(p) \neq 0$$

通过精心设计迭代函数 $g(x)$ 中的参数，可以使其更多的低阶导数为零，从而提高收敛阶。例如，考虑一个用于计算 $A^{1/3}$ 的迭代族 ：
$$x_{n+1} = g(x_n) = x_n \left( \frac{x_n^3 + \alpha A}{\beta x_n^3 + A} \right)$$
为了使 $p=A^{1/3}$ 成为[不动点](@entry_id:156394)，必须有 $\alpha=\beta$。在此约束下，通过复杂的求导和分析，可以发现当选择 $\alpha=2$ 时，不仅有 $g'(p)=0$，还有 $g''(p)=0$，但 $g'''(p) \neq 0$。这意味着通过选择最优参数 $\alpha=2$，该[迭代法](@entry_id:194857)可以达到三阶收敛，这是一个非常快的收敛速度。

### 难点与实际应用考量

#### 中性[不动点](@entry_id:156394)与吸引盆
当 $|g'(x^*)|=1$ 时，情况变得复杂。线性分析失效，收敛性取决于[泰勒展开](@entry_id:145057)中的[非线性](@entry_id:637147)项。
以迭代 $x_{n+1} = g(x_n) = x_n^2 - x_n + 1$ 为例 。其唯一[不动点](@entry_id:156394)是 $x^*=1$。在[不动点](@entry_id:156394)处，导数 $g'(x) = 2x-1$ 的值为 $g'(1)=1$，这是一个中性[不动点](@entry_id:156394)。
为了分析其行为，我们考察误差 $e_n = x_n - 1$ 的演化：
$$e_{n+1} = x_{n+1} - 1 = (x_n^2 - x_n + 1) - 1 = (e_n+1)^2 - (e_n+1) = e_n^2 + e_n = e_n(1+e_n)$$
-   如果初始值 $x_0 > 1$（即 $e_0 > 0$），那么 $e_1 = e_0(1+e_0) > e_0$，序列将单调递增并迅速发散到无穷大。
-   如果初始值 $0  x_0  1$（即 $-1  e_0  0$），那么 $e_n$ 将是一个负数序列，且 $e_{n+1} = e_n(1+e_n) > e_n$（因为 $1+e_n  1$），所以序列单调递增并有上界0，故收敛到0。
-   如果 $x_0 = 0$（即 $e_0 = -1$），则 $x_1=1$，序列收敛。
-   如果 $x_0  0$（即 $e_0  -1$），则 $e_1$ 会变为正数，之后便发散。

综上所述，只有当初始值 $x_0$ 位于**[吸引盆](@entry_id:174948)**（basin of attraction）$[0, 1]$ 内时，迭代才会收敛到[不动点](@entry_id:156394)1。这个例子说明，中性[不动点](@entry_id:156394)的收敛行为可能非常微妙，并且[收敛域](@entry_id:269722)可能很有限。

#### 有限精度计算的影响
在真实的计算机上，所有计算都是以有限精度（[浮点数](@entry_id:173316)）进行的。这为迭代过程引入了[舍入误差](@entry_id:162651)，即使是理论上收敛的迭代，也无法无限地逼近真值。

考虑一个收敛的迭代过程 $p_{n+1}=g(p_n)$，如 $g(x)=\sqrt{x}$ 求解[不动点](@entry_id:156394) $p=1$ 。其导数 $g'(1) = 1/2$，是收敛的。在计算机上，每一步的计算结果可以建模为 $\hat{p}_{n+1} = g(\hat{p}_n)(1+\delta_n)$，其中 $\delta_n$ 是相对舍入误差，其大小受限于[机器精度](@entry_id:756332) $\epsilon_{mach}$，即 $|\delta_n| \le \epsilon_{mach}$。

令 $\hat{e}_n = \hat{p}_n - 1$ 为计算误差。其演化关系近似为：
$$\hat{e}_{n+1} \approx g'(\hat{p}_n)\hat{e}_n + g(\hat{p}_n)\delta_n$$
当 $\hat{p}_n$ 接近1时，$g'(\hat{p}_n) \approx 1/2$ 且 $g(\hat{p}_n) \approx 1$，因此：
$$|\hat{e}_{n+1}| \le \frac{1}{2}|\hat{e}_n| + \epsilon_{mach}$$
在迭代初期，由于 $|\hat{e}_n|$ 较大，收缩项 $\frac{1}{2}|\hat{e}_n|$ 占主导，误差不断减小。然而，当 $|\hat{e}_n|$ 减小到与 $\epsilon_{mach}$ 同一量级时，新的[舍入误差](@entry_id:162651) $\epsilon_{mach}$ 开始变得不可忽略。最终，迭代序列将不再稳定地逼近[真值](@entry_id:636547)，而是在一个“数值稳定区间”[内波](@entry_id:261048)动。这个区间的半宽度 $E$ 可以通过平衡误差的缩小和引入来估计：
$$E \approx \frac{1}{2}E + \epsilon_{mach} \quad \implies \quad \frac{1}{2}E \approx \epsilon_{mach} \quad \implies \quad E \approx 2\epsilon_{mach}$$
这意味着，由于[浮点误差](@entry_id:173912)的存在，即使使用理论上收敛的方法，我们能达到的最终精度也存在一个由[机器精度](@entry_id:756332)决定的基本极限。这是从理论数学转向实际数值计算时必须面对的重要现实。
