## 引言
[偏微分方程](@entry_id:141332)（PDEs）是描述从热量传导、[流体运动](@entry_id:182721)到金融市场波动的各种物理现象的通用语言。然而，这些方程大多形式复杂，难以求得解析解。[有限差分法](@entry_id:147158)（Finite Difference Method, FDM）应运而生，它作为一种基础且强大的数值技术，通过将连续的方程和定义域离散化，把求解微分方程的问题转化为计算机可以处理的代数方程组问题，从而在科学与工程计算中扮演着不可或缺的角色。

然而，从连续的PDE到离散的数值解，这一过程并非没有代价。关键的挑战在于如何在保证[计算效率](@entry_id:270255)的同时，确保数值解能够忠实地反映原始物理过程的真实行为。这涉及到对离散化引入的误差的深刻理解，以及对数值[算法稳定性](@entry_id:147637)的严格控制。本文旨在系统性地解决这一知识鸿沟，带领读者从零开始，全面掌握有限差分法的理论精髓与实践技巧。

在接下来的内容中，我们将分三步深入探索有限差分法的世界。首先，在“原理与机制”章节中，我们将奠定理论基石，详细剖析如何构建差分格式、分析其误差来源，并建立保证解收敛的稳定性条件。随后，在“应用与[交叉](@entry_id:147634)学科联系”章节中，我们将跨越学科界限，展示[有限差分法](@entry_id:147158)在工程、物理、生物、金融乃至机器学习等前沿领域的广泛应用，让你看到理论如何转化为解决实际问题的强大工具。最后，在“动手实践”部分，你将有机会通过具体的编程练习，亲手实现并验证关键的数值算法，将理论知识内化为实践能力。现在，让我们从构建[有限差分法](@entry_id:147158)的第一块基石——离散化开始我们的学习之旅。

## 原理与机制

在“引言”章节中，我们介绍了[偏微分方程](@entry_id:141332)（PDE）有限差分法（FDM）的基本思想。本章将深入探讨其核心原理与机制。我们将从构建离散近似的基础出发，系统地分析由此产生的误差的性质，并建立保证数值解收敛于真实解的严格条件。最后，我们将探讨如何将这些基本原理应用于更复杂、更贴近实际的物理问题，例如多维问题、[非线性](@entry_id:637147)效应以及不规则的几何边界。

### 离散化：从导数到[差分方程](@entry_id:262177)

[有限差分法](@entry_id:147158)的基石是用离散点上的函数值组合来近似连续的导数。这一过程最根本的工具是[泰勒级数展开](@entry_id:138468)。对于一个在点 $x_i$ 附近足够光滑的函数 $u(x)$，其在邻近点 $x_{i+1} = x_i + \Delta x$ 和 $x_{i-1} = x_i - \Delta x$ 的值可以展开为：

$u(x_{i+1}) = u(x_i) + \Delta x u'(x_i) + \frac{(\Delta x)^2}{2} u''(x_i) + \frac{(\Delta x)^3}{6} u'''(x_i) + \mathcal{O}((\Delta x)^4)$

$u(x_{i-1}) = u(x_i) - \Delta x u'(x_i) + \frac{(\Delta x)^2}{2} u''(x_i) - \frac{(\Delta x)^3}{6} u'''(x_i) + \mathcal{O}((\Delta x)^4)$

通过对这两个展开式进行代数操作，我们可以构造出导数的近似。例如，两式相减再除以 $2\Delta x$，可以得到**[中心差分](@entry_id:173198)**格式来近似一阶导数：

$\frac{u(x_{i+1}) - u(x_{i-1})}{2\Delta x} = u'(x_i) + \frac{(\Delta x)^2}{6} u'''(x_i) + \dots = u'(x_i) + \mathcal{O}((\Delta x)^2)$

而将两式相加再进行整理，则可以得到近似[二阶导数](@entry_id:144508)的[中心差分格式](@entry_id:747203)：

$\frac{u(x_{i+1}) - 2u(x_i) + u(x_{i-1})}{(\Delta x)^2} = u''(x_i) + \frac{(\Delta x)^2}{12} u^{(4)}(x_i) + \dots = u''(x_i) + \mathcal{O}((\Delta x)^2)$

在这些推导中，我们忽略的高阶项被称为**[局部截断误差](@entry_id:147703)（Local Truncation Error, LTE）**。它是将微分算子应用于精确解和将差分算子应用于精确解之间的差异。对于上述[中心差分格式](@entry_id:747203)，由于其[局部截断误差](@entry_id:147703)与 $(\Delta x)^2$ 成正比，我们称它们是**二阶精确**的。这意味着当网格间距 $\Delta x$ 减半时，误差大约会减小到原来的四分之一，这是一种理想的收敛行为。

然而，对网格均匀性的依赖是这些标准公式的一个重要特性。考虑在[非均匀网格](@entry_id:752607)上逼近[二阶导数](@entry_id:144508)的情形，其中点 $x_i$ 的左邻居和右邻居距离不等，分别为 $h_1 = x_i - x_{i-1}$ 和 $h_2 = x_{i+1} - x_i$ 。通过求解一个要求公式对常数、线性和二次多项式都精确的[线性系统](@entry_id:147850)，可以得到一个更通用的三点差分公式：

$D^2 u(x_i) = \frac{2}{h_1(h_1+h_2)}u(x_{i-1}) - \frac{2}{h_1h_2}u(x_i) + \frac{2}{h_2(h_1+h_2)}u(x_{i+1})$

通过泰勒展开分析这个公式的[局部截断误差](@entry_id:147703) $\tau(x_i) = D^2 u(x_i) - u''(x_i)$，我们发现其[主导项](@entry_id:167418)为：

$\tau(x_i) = \frac{h_2 - h_1}{3} u'''(x_i) + \mathcal{O}((h_1+h_2)^2)$

这个结果揭示了一个关键点：在[非均匀网格](@entry_id:752607)上 ($h_1 \neq h_2$)，该差分格式的精度会降至一阶，即误差与网格间距之差成正比。只有在均匀网格 ($h_1 = h_2 = \Delta x$) 的特殊情况下，这个主导误差项才会消失，从而恢复[二阶精度](@entry_id:137876)，此时截断误差变为 $\mathcal{O}((\Delta x)^2)$。这强调了在实践中，保持网格的光滑和均匀对于获得高精度解的重要性。

### [数值误差](@entry_id:635587)的剖析：数值扩散与数值频散

用差分算子替换微分算子后，我们得到的离散方程在何种程度上模拟了原始的[偏微分方程](@entry_id:141332)？一个强有力的分析工具是**修正方程（Modified Equation）**分析。其思想是通过[泰勒展开](@entry_id:145057)，将离散的差分算子重新表达为一系列连续的微分算子，从而揭示出我们的[数值格式](@entry_id:752822)实际上在求解哪个“修正”了的PDE。

我们以一维线性[平流方程](@entry_id:144869) $u_t + a u_x = 0$（其中 $a > 0$ 为常数[波速](@entry_id:186208)）为例来说明两种最典型的数值误差类型。

首先，考虑用一阶**迎风格式（upwind scheme）**对空间导数进行[半离散化](@entry_id:163562)：$u_t + a \frac{u_i - u_{i-1}}{\Delta x} = 0$。将差分项在 $x_i$ 点展开，我们得到：

$a \frac{u_i - u_{i-1}}{\Delta x} = a \left( u_x - \frac{\Delta x}{2} u_{xx} + \mathcal{O}((\Delta x)^2) \right)$

代入半离散方程，得到修正方程为：

$u_t + a u_x = a \frac{\Delta x}{2} u_{xx} + \dots$

原始的[平流方程](@entry_id:144869)是无损耗的，波形应保持不变向前传播。然而，修正方程的右侧出现了一个[二阶导数](@entry_id:144508)项，其形式与物理[扩散](@entry_id:141445)（如[热传导](@entry_id:147831)）中的项完全一样。这个由离散化引入的、非物理的[扩散](@entry_id:141445)效应被称为**数值扩散（numerical diffusion）**。它的系数 $\nu = a \Delta x / 2$ 正比于网格尺寸 $\Delta x$。这种效应会导致数值解中的尖锐梯度（如激波或[接触间断](@entry_id:194702)）被抹平或模糊化，就好像它们在经历物理[扩散](@entry_id:141445)一样。

接下来，我们考察[二阶中心差分](@entry_id:170774)格式：$u_t + a \frac{u_{i+1} - u_{i-1}}{2 \Delta x} = 0$。同样，展开差分项得到：

$a \frac{u_{i+1} - u_{i-1}}{2 \Delta x} = a \left( u_x + \frac{(\Delta x)^2}{6} u_{xxx} + \mathcal{O}((\Delta x)^4) \right)$

此时，修正方程变为：

$u_t + a u_x = -a \frac{(\Delta x)^2}{6} u_{xxx} + \dots$

这一次，主导误差项是一个三阶导数，它与物理[扩散](@entry_id:141445)无关。这种形式的项会引起**数值频散（numerical dispersion）**。在物理学中，频散指的是不同波长的波以不同速度传播的现象。数值频散效应类似，它导致数值解中不同频率的分量以错误的速度传播，从而在光滑的解附近产生非物理的、高频的[振荡](@entry_id:267781)或伪影。

修正方程分析告诉我们，即使一个格式是收敛的，其解的质量也可能因其主导的[数值误差](@entry_id:635587)类型而大相径庭。[迎风格式](@entry_id:756374)因其数值扩散性而倾向于产生平滑但模糊的解，而[中心差分格式](@entry_id:747203)则可能产生锐利但带有[振荡](@entry_id:267781)的解。理解这些特性对于选择合适的数值格式至关重要。

### 稳定性：防止误差的爆炸性增长

在数值计算中，即使[局部截断误差](@entry_id:147703)很小，这些在每一步引入的微小误差（包括[舍入误差](@entry_id:162651)）也可能在迭代过程中累积并指数级增长，最终完全淹没真实的解。**[数值稳定性](@entry_id:146550)**是确保这种情况不会发生的基本要求。一个稳定的格式能够抑制误差的增长。

#### CFL 条件

对于求解双曲型PDE（如平流方程）的显式差分格式，一个基本而直观的稳定性必要条件是**[Courant-Friedrichs-Lewy (CFL) 条件](@entry_id:747986)**。该条件可以通过比较PDE的物理[依赖域](@entry_id:160270)和差分格式的[数值依赖域](@entry_id:163312)来理解。

考虑平流方程 $u_t + c u_x = 0$。其解的特性是信息沿特征线 $x - ct = \text{const}$ 传播。这意味着在 $(x_j, t_{n+1})$ 处的精确解，其值完全由初始时刻 ($t=0$) 在一个特定点 $x_0 = x_j - c t_{n+1}$ 的值决定。这个点就是该精确解的**物理[依赖域](@entry_id:160270)**。

现在考察一个显式差分格式，例如前向时间中心空间（FTCS）格式：$u_j^{n+1} = u_j^n - \frac{c \Delta t}{2 \Delta x} (u_{j+1}^n - u_{j-1}^n)$。$u_j^{n+1}$ 的值依赖于前一时刻的三个点 $u_{j-1}^n, u_j^n, u_{j+1}^n$。追溯到初始时刻 $t=0$， $u_j^{n+1}$ 的值会受到初始数据在一个区间 $[x_j - (n+1)\Delta x, x_j + (n+1)\Delta x]$ 内所有点的影响。这个区间就是该数值解的**[数值依赖域](@entry_id:163312)**。

CFL 原理指出：为了让[数值格式](@entry_id:752822)能够收敛，其[数值依赖域](@entry_id:163312)必须包含PDE的物理[依赖域](@entry_id:160270)。直观上，如果决定真实解的关键信息位于[数值格式](@entry_id:752822)“看不到”的地方，那么数值解就不可能正确地逼近真实解。对于平流方程，这意味着：

$|x_j - c t_{n+1}| \leq |x_j - (n+1)\Delta x|$  (假设 $c>0$)

化简后得到：

$c \Delta t \leq \Delta x \quad \text{或} \quad \sigma = \frac{c \Delta t}{\Delta x} \leq 1$

其中 $\sigma$ 被称为 **Courant 数**。CFL 条件为显式格式的时间步长 $\Delta t$ 设定了一个上限。值得注意的是，虽然[CFL条件](@entry_id:178032)是必要的，但它并不总是充分的（例如，对于[平流方程](@entry_id:144869)，[FTCS格式](@entry_id:146585)即使满足[CFL条件](@entry_id:178032)也是无条件不稳定的）。

#### [冯·诺依曼稳定性分析](@entry_id:145718)

对于[常系数](@entry_id:269842)线性PDE，**冯·诺依曼（von Neumann）[稳定性分析](@entry_id:144077)**是一种更形式化、更强大的工具。其核心思想是将数值解的[误差分解](@entry_id:636944)为一系列傅里叶模式，并考察每个模式在时间步进中的增长或衰减。

我们假设解的一个典型傅里叶模式具有形式 $u_j^n = G^n e^{i k j \Delta x}$，其中 $k$ 是[波数](@entry_id:172452)，$G$ 是**[放大因子](@entry_id:144315)（amplification factor）**，它表示该模式在一个时间步长内的复数增益。将这个模式代入[差分方程](@entry_id:262177)，就可以解出 $G$ 作为波数 $k$（或相位角 $\theta = k \Delta x$）的函数。

为了保证所有傅里叶模式的误差都不会增长，[放大因子](@entry_id:144315)的模必须对所有可能的波数都满足 $|G| \leq 1$。

让我们以一个包含反应项的一维线性[反应-扩散方程](@entry_id:170319)为例：$u_t = D u_{xx} + \kappa u$ 。采用[FTCS格式](@entry_id:146585)进行离散化：

$\frac{u_j^{n+1} - u_j^n}{\Delta t} = D \frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{(\Delta x)^2} + \kappa u_j^n$

代入傅里叶模式后，经过化简可得放大因子 $G$：

$G = 1 + \alpha (e^{i\theta} - 2 + e^{-i\theta}) + \beta = 1 + \beta + 2\alpha(\cos\theta - 1) = 1 + \beta - 4\alpha\sin^2(\frac{\theta}{2})$

这里 $\alpha = D \Delta t / (\Delta x)^2$ 是[扩散](@entry_id:141445)数，$\beta = \kappa \Delta t$ 是反应数。稳定性条件 $|G| \leq 1$ 等价于 $-1 \leq G \leq 1$。

1.  上界 $G \leq 1$：$1 + \beta - 4\alpha\sin^2(\frac{\theta}{2}) \leq 1$，这意味着 $\beta \leq 4\alpha\sin^2(\frac{\theta}{2})$。
    - 如果反应项是增长性的（$\kappa > 0$, 因此 $\beta > 0$），这个条件不可能对所有波数都成立（特别是当 $\theta \to 0$ 时，右侧趋于零）。因此，对于任何增长性反应，[FTCS格式](@entry_id:146585)都是**无条件不稳定**的。
    - 如果反应项是耗散性的（$\kappa \le 0$, 因此 $\beta \le 0$），由于 $\alpha > 0$，这个条件总是成立的。
2.  下界 $G \geq -1$ (仅在 $\kappa \le 0$ 时有意义)：$1 + \beta - 4\alpha\sin^2(\frac{\theta}{2}) \geq -1$，化简得 $4\alpha\sin^2(\frac{\theta}{2}) \leq 2 + \beta$。

这个不等式必须对所有波数（即所有 $\theta$）都成立。最严格的限制发生在 $\sin^2(\frac{\theta}{2})$ 取其最大值 1 时。因此，我们得到了最终的稳定性条件：

$4\alpha \leq 2 + \beta \quad \implies \quad \alpha \leq \frac{2 + \beta}{4}$

这个结果揭示了与纯[扩散](@entry_id:141445)问题（$\beta=0, \alpha \le 1/2$）相比，一个[耗散性](@entry_id:162959)的反应项（$\beta  0$）会使 $2+\beta  2$，从而使得对时间步长的限制**更加严格**。这与物理直觉相符：一个快速衰减的物理过程需要更小的时间步长来准确解析。

### 收敛性：Lax 等价定理

我们最终的目标是确保当网格无限加密时（$\Delta x, \Delta t \to 0$），数值解能够趋近于PDE的真实解。这一性质被称为**收敛性（convergence）**。**Lax 等价定理（Lax Equivalence Theorem）**为我们提供了理论上的最终保证。

该定理指出，对于一个适定（well-posed）的线性初值问题，一个**相容（consistent）**的[有限差分格式](@entry_id:749361)是收敛的，当且仅当它是**稳定（stable）**的。

**相容性 + 稳定性 $\iff$ 收敛性**

这个定理是数值[PDE理论](@entry_id:189232)的基石。它将复杂的收敛性[问题分解](@entry_id:272624)为两个相对更容易验证的部分：
*   **相容性**：[局部截断误差](@entry_id:147703)随着网格尺寸趋于零而趋于零。这本质上是检验我们的[差分方程](@entry_id:262177)是否确实在逼近我们想要解的那个PDE。
*   **稳定性**：数值解在误差存在的情况下保持有界。

[Lax等价定理](@entry_id:139112)的应用不仅限于标量方程，它对于PDE系统同样至关重要。例如，在线性化的浅水波[方程组](@entry_id:193238)中，稳定性分析必须针对整个耦合系统进行，而不能将各个方程孤立地分开处理。这通常涉及分析一个**[放大矩阵](@entry_id:746417)（amplification matrix）** $\mathbf{G}$，稳定性要求其谱半径（最大[特征值](@entry_id:154894)的模） $\rho(\mathbf{G}) \leq 1$。对于更复杂的系统，如可以写成对称双曲形式的系统，还可以使用能量方法，在由系统物理性质导出的特定范数下证明稳定性。只要在适当的范数下证明了相容性和稳定性，[Lax等价定理](@entry_id:139112)就保证了收敛性。

### 应用于复杂问题

掌握了离散化、[误差分析](@entry_id:142477)和稳定性的基本原理后，我们可以将它们应用于更具挑战性的场景。

#### 多维问题与矩阵结构

将有限差分法扩展到多维空间是直接的，但这会带来巨大的计算挑战。以二维泊松方程 $-\Delta u = - (u_{xx} + u_{yy}) = f$ 为例。在一个 $N_x \times N_y$ 的内部网格点上，我们可以使用标准的**[五点模板](@entry_id:174268)**来离散化[拉普拉斯算子](@entry_id:146319)：

$\frac{u_{i-1,j} - 2u_{i,j} + u_{i+1,j}}{h_x^2} + \frac{u_{i,j-1} - 2u_{i,j} + u_{i,j+1}}{h_y^2} = -f_{i,j}$

这为每个内部网格点提供了一个[线性方程](@entry_id:151487)。挑战在于如何将这 $N_x N_y$ 个[方程组](@entry_id:193238)织成一个单一的线性系统 $A\mathbf{u} = \mathbf{b}$。一个常见的策略是**字典序（lexicographic ordering）**或[行主序](@entry_id:634801)，即逐行将二维网格上的未知数 $u_{i,j}$ [排列](@entry_id:136432)成一个长向量 $\mathbf{u}$。

这种排序方式产生了一个具有特定结构的[稀疏矩阵](@entry_id:138197) $A$。对于点 $(i,j)$，它与其左右邻居 $(i\pm 1, j)$ 的耦合对应于矩阵主对角线附近的元素。然而，它与上下邻居 $(i, j\pm 1)$ 的耦合，在长向量中的索引相差 $N_x$。这导致矩阵 $A$ 具有**块三对角（block-tridiagonal）**结构。矩阵 $A$ 由 $N_y \times N_y$ 个块组成，每个块的大小为 $N_x \times N_x$。对角块本身是三对角矩阵，代表了x方向的耦合；而非对角块是对角矩阵，代表了y方向的耦合。理解这种结构对于选择高效的线性代数求解器（如迭代法或稀疏直接法）至关重要。对于[泊松方程](@entry_id:143763)，所得到的矩阵通常是**[对称正定](@entry_id:145886)**的，这为求解提供了有利的性质。

#### [隐式方法](@entry_id:137073)、线方法与[刚性问题](@entry_id:142143)

对于[演化方程](@entry_id:268137)（抛物型或双曲型），显式方法（如FTCS）的稳定性条件常常对时间步长 $\Delta t$ 施加严格的限制。特别是对于[扩散](@entry_id:141445)问题，$\Delta t$ 必须与 $(\Delta x)^2$ 成比例。这意味着如果要将空间分辨率提高一倍，就必须将时间步长减小到四分之一，计算成本会急剧增加。

当问题中存在极大或极小的物理参数时，情况会变得更糟。考虑[非线性](@entry_id:637147)[扩散方程](@entry_id:170713) $u_t = \partial_x (D(u) \partial_x u)$ 。其显式格式的稳定性条件大致为 $\Delta t \le \frac{(\Delta x)^2}{2 D_{\max}}$，其中 $D_{\max}$ 是[扩散](@entry_id:141445)系数的最大值。如果 $D(u)$ 在某些区域非常大，这个稳定性约束会迫使我们使用一个远小于保证精度所需要的 $\Delta t$。这种由于稳定性而非精度要求所导致的对时间步长的极端限制，是**刚性（stiffness）**问题的标志。

为了克服刚性，我们需要使用**隐式方法（implicit methods）**。这些方法在计算 $t^{n+1}$ 时刻的解时，使用了 $t^{n+1}$ 时刻的未知值。例如，**Crank-Nicolson** 格式通过对空间导数在 $t^n$ 和 $t^{n+1}$ 时刻进行平均，从而获得时间上的二阶精度和[无条件稳定性](@entry_id:145631)（对于线性问题）。

一个优雅的视角是**线方法（Method of Lines, MOL）**。该方法首先只在空间上进行离散化，将一个PDE转化为一个大型的常微分方程（ODE）组 $U'(t) = F(U(t))$，其中 $U(t)$ 是所有空间网格点上解的向量。然后，我们可以应用任何为ODE设计的[数值积分器](@entry_id:752799)来求解这个系统。例如，将梯形法则（一种二阶精确的隐式ODE求解器）应用于由[热方程](@entry_id:144435)[半离散化](@entry_id:163562)得到的ODE系统，我们得到的全离散格式与经典的[Crank-Nicolson格式](@entry_id:147733)完全相同。这个视角将PDE的求解分解为[空间离散化](@entry_id:172158)和时间积分两个独立步骤，为设计和分析复杂的[数值格式](@entry_id:752822)提供了一个强大的框架。

#### 处理复杂的物理特性

真实世界的问题往往包含不连续的材料属性或不规则的几何形状，标准的[有限差分法](@entry_id:147158)需要进行巧妙的推广才能应对。

*   **不连续系数**：考虑一个[稳态扩散](@entry_id:154663)问题 $-\frac{d}{dx}(\kappa(x) \frac{du}{dx}) = f(x)$，其中[扩散](@entry_id:141445)系数 $\kappa(x)$ 在[材料界面](@entry_id:751731)处发生跳跃。物理上，在界面处，解 $u(x)$ 本身是连续的，但其通量 $F = -\kappa(x) \frac{du}{dx}$ 也是连续的。一个简单的算术平均来计算界面上的[有效扩散系数](@entry_id:183973) $\kappa_{i+1/2}$ 会违反通量连续性，并导致精度损失。正确的做法是使用**调和平均（harmonic averaging）**：
    $\kappa_{i+1/2} = \left( \frac{1}{2} \left( \frac{1}{\kappa_i} + \frac{1}{\kappa_{i+1}} \right) \right)^{-1} = \frac{2\kappa_i \kappa_{i+1}}{\kappa_i + \kappa_{i+1}}$
    这种平均方式能够在线性情形下精确地满足通量连续性条件，从而在界面附近保持数值解的准确性。

*   **弯曲边界**：标准的[有限差分法](@entry_id:147158)天然适用于矩形区域。当处理带有弯曲边界的复杂几何时，一种强大的技术是**[切割单元法](@entry_id:172250)（cut-cell method）**。这种方法本质上是有限差分法和有限体积法的结合。其核心思想是在被边界切割的网格单元上，直接对PDE进行积分，然后应用[散度定理](@entry_id:143110)：
    $\int_C (-\Delta u) \, dA = \int_C f \, dA \implies \oint_{\partial C} (-\nabla u \cdot \boldsymbol{n}) \, ds = \int_C f \, dA$
    这里 $C$ 是被切割的单元，$\partial C$ 是它的边界，包括笔直的网格线[部分和](@entry_id:162077)弯曲的物理边界部分。这个方程表达了穿过单元边界的总通量等于单元内部源的总和。通过为每个边界段上的通量构造精确的近似（例如，对弯曲边界上的通量使用给定的边界条件），我们可以为靠近边界的节点建立一个修正后的差分模板。这种方法能够保持整个计算域的守恒性，并可以实现[高阶精度](@entry_id:750325)，是处理复杂几何问题的一种有效途径。

通过本章的学习，我们不仅掌握了构建[有限差分格式](@entry_id:749361)的基本技术，更重要的是，我们建立了分析和理解其行为的理论框架。从[截断误差](@entry_id:140949)的性质到稳定性的严格条件，再到处理多维、[非线性](@entry_id:637147)和复杂几何的策略，这些原理和机制共同构成了使用[有限差分法](@entry_id:147158)成功求解偏微分方程的基石。
