## 原理与机制

在对随机现象进行[数学建模](@entry_id:262517)时，核心任务是描述一个变量可能取值的概率。[随机变量](@entry_id:195330) (random variable) 的概念为我们提供了一个将随机试验的结果映射到实数的框架，而其概率行为则由一个称为**[累积分布函数](@entry_id:143135) (Cumulative Distribution Function, CDF)** 的强大工具完全刻画。本章旨在深入探讨[累积分布函数](@entry_id:143135)的原理、基本性质及其在理论与计算科学中的关键机制。

### 累积分布函数的定义与构建

[随机变量](@entry_id:195330) $X$ 的**[累积分布函数](@entry_id:143135)**，记为 $F_X(x)$，定义为该[随机变量](@entry_id:195330)取值不大于某个实数 $x$ 的概率：

$$ F_X(x) = P(X \le x) $$

对于任意实数 $x$，$F_X(x)$ 都是一个明确定义的概率值。因此，CDF 是一个定义在整个[实数轴](@entry_id:147286) $\mathbb{R}$ 上的函数，其值域为 $[0, 1]$。这个函数之所以重要，是因为它完全捕捉了[随机变量](@entry_id:195330) $X$ 的所有概率信息。一旦我们知道了 $F_X(x)$，原则上就可以回答关于 $X$ 的任何概率问题。

让我们通过一个具体的例子来理解如何构建和使用 CDF。假设一个[离散随机变量](@entry_id:163471) $X$ 只能取三个值：$-2$、$0$ 和 $\sqrt{3}$。其取各个值的概率由一个正常数 $c$ 决定，分别为 $P(X=-2) = c$，$P(X=0) = 2c$ 和 $P(X=\sqrt{3}) = 3c$。

根据[概率论公理](@entry_id:198155)，所有可能结果的概率之和必须为 1。因此，我们可以确定常数 $c$ 的值：
$$ c + 2c + 3c = 6c = 1 \implies c = \frac{1}{6} $$
于是，我们得到具体的概率：$P(X=-2) = \frac{1}{6}$，$P(X=0) = \frac{2}{6} = \frac{1}{3}$，以及 $P(X=\sqrt{3}) = \frac{3}{6} = \frac{1}{2}$。

现在，我们可以根据定义 $F_X(x) = P(X \le x)$ 来构建 CDF。我们需要考虑 $x$ 在实数轴上移动时，事件 $\{X \le x\}$ 所包含的结果如何变化：

-   当 $x  -2$ 时，没有任何可能的结果小于或等于 $x$，所以 $F_X(x) = P(\emptyset) = 0$。

-   当 $-2 \le x  0$ 时，唯一小于或等于 $x$ 的结果是 $-2$。因此，$F_X(x) = P(X \le x) = P(X=-2) = \frac{1}{6}$。

-   当 $0 \le x  \sqrt{3}$ 时，小于或等于 $x$ 的结果包括 $-2$ 和 $0$。因此，$F_X(x) = P(X \le x) = P(X=-2) + P(X=0) = \frac{1}{6} + \frac{1}{3} = \frac{1}{2}$。

-   当 $x \ge \sqrt{3}$ 时，所有可能的结果都小于或等于 $x$。因此，$F_X(x) = P(X \le x) = P(X=-2) + P(X=0) + P(X=\sqrt{3}) = 1$。

将这些片段组合起来，我们得到了一个完整的 CDF，它是一个**阶梯函数 (step function)**：
$$
F_X(x) = \begin{cases}
0  \text{if } x  -2 \\
\frac{1}{6}  \text{if } -2 \le x  0 \\
\frac{1}{2}  \text{if } 0 \le x  \sqrt{3} \\
1  \text{if } x \ge \sqrt{3}
\end{cases}
$$
这个函数在每个可能的取值点 $-2, 0, \sqrt{3}$ 处发生跳跃，跳跃的高度恰好等于该点的概率。例如，在 $x=0$ 处，函数值从 $\frac{1}{6}$ 跳跃到 $\frac{1}{2}$，跳跃高度为 $\frac{1}{2} - \frac{1}{6} = \frac{1}{3} = P(X=0)$。

利用这个构建好的 CDF，我们可以轻易地计算出[随机变量](@entry_id:195330)落在特定区间的概率。例如，计算 $F_X(-1)$，即 $P(X \le -1)$，根据我们的分段定义，$-1$ 属于区间 $[-2, 0)$，所以 $F_X(-1) = \frac{1}{6}$ [@problem_id:1416770]。

### CDF 的公理化性质

并非任何函数都可以作为某个[随机变量](@entry_id:195330)的 CDF。一个函数 $F: \mathbb{R} \to [0, 1]$ 若要成为一个合法的 CDF，必须满足以下三个基本性质：

1.  **非递减性 (Non-decreasing)**：对任意 $x_1  x_2$，必有 $F(x_1) \le F(x_2)$。这个性质源于概率的单调性。因为事件 $\{X \le x_1\}$ 是事件 $\{X \le x_2\}$ 的[子集](@entry_id:261956)，所以前者的概率不能超过后者。

2.  **正确的渐近极限 (Asymptotic Limits)**：$\lim_{x\to-\infty} F(x) = 0$ 且 $\lim_{x\to+\infty} F(x) = 1$。前者表示当 $x$ 趋于负无穷时，事件 $\{X \le x\}$ 变为空事件，其概率为 0。后者表示当 $x$ 趋于正无穷时，事件 $\{X \le x\}$ 变为整个样本空间，其概率为 1。

3.  **[右连续性](@entry_id:170543) (Right-continuity)**：对任意 $x \in \mathbb{R}$，必有 $\lim_{h\to 0^+} F(x+h) = F(x)$。这个性质是一个约定，但至关重要。它确保了 CDF 在一个点的函数值 $F(x_0)$ 包含了 $X=x_0$ 这一事件本身的概率。

让我们通过一个反例来理解[右连续性](@entry_id:170543)的重要性。考虑函数 $A(x) = \begin{cases} 0  \text{if } x \le 0 \\ 1  \text{if } x > 0 \end{cases}$。它满足非递减性和正确的渐近极限。然而，在 $x=0$ 点，我们有 $A(0)=0$，但从右侧逼近时，$\lim_{h\to 0^+} A(0+h) = \lim_{h\to 0^+} 1 = 1$。由于 $A(0) \neq \lim_{h\to 0^+} A(0+h)$，该函数在 $x=0$ 处是左连续而非右连续的，因此它不是一个合法的 CDF。相比之下，函数 $B(x) = \begin{cases} 0  \text{if } x  0 \\ 1  \text{if } x \ge 0 \end{cases}$ 在 $x=0$ 处是右连续的，它是一个合法的 CDF，描述了一个在 $x=0$ 点取值为 1 的[随机变量](@entry_id:195330) [@problem_id:1416747]。

这些公理化性质为我们提供了一套检验标准。假设一位[统计物理学](@entry_id:142945)家提出了一个描述粒子位置的潜在 CDF [@problem_id:1416758]：
$$
F(x; \alpha) = \begin{cases}
0  \text{for } x  0 \\
\frac{x}{\pi} + \alpha (x - \sin(x))  \text{for } 0 \leq x  \pi \\
1  \text{for } x \geq \pi
\end{cases}
$$
为了使这个函数成为一个合法的 CDF，参数 $\alpha$ 必须满足什么条件呢？
首先，极限性质已经由分段定义保证。[右连续性](@entry_id:170543)在 $x=0$ 处也满足，因为 $F(0; \alpha) = 0$。我们需要关注的是非递减性和在连接点 $x=\pi$ 处的行为。

-   **非递减性**：在 $(0, \pi)$ 区间内，函数必须是非递减的。如果函数可导，这意味着其导数必须非负。
    $$ F'(x; \alpha) = \frac{1}{\pi} + \alpha (1 - \cos x) \ge 0 $$
    由于 $x \in (0, \pi)$ 时，$1-\cos x$ 的取值范围是 $(0, 2]$。如果 $\alpha \ge 0$，该不等式显然成立。如果 $\alpha  0$，则不等式最难满足的情况出现在 $1-\cos x$ 取最大值 2 时（即 $x \to \pi^-$）。此时需要 $\frac{1}{\pi} + 2\alpha \ge 0$，即 $\alpha \ge -\frac{1}{2\pi}$。

-   **在连接点 $x=\pi$ 处的行为**：为了保证整体的非递减性，从左侧逼近 $\pi$ 时的函数值不能超过 $F(\pi)=1$。
    $$ \lim_{x \to \pi^-} F(x; \alpha) = \lim_{x \to \pi^-} \left( \frac{x}{\pi} + \alpha (x - \sin x) \right) = 1 + \alpha \pi $$
    因此，我们需要 $1 + \alpha \pi \le 1$，这意味着 $\alpha \pi \le 0$，即 $\alpha \le 0$。

结合两个条件，我们发现参数 $\alpha$ 必须在闭区间 $[-\frac{1}{2\pi}, 0]$ 内。这个例子清晰地展示了如何运用 CDF 的公理化性质来约束一个数学模型。

### CDF 的结构：跳跃、斜率与分解

CDF 的几何形态揭示了[随机变量](@entry_id:195330)的深层结构。

#### 跳跃与离散型[随机变量](@entry_id:195330)
CDF 的不连续点，即**跳跃点**，对应着[随机变量](@entry_id:195330)取特定值的正概率。在任意一点 $x_0$，CDF 的跳跃大小定义为：
$$ j(x_0) = F(x_0) - \lim_{x \to x_0^-} F(x) $$
这个跳跃值恰好等于[随机变量](@entry_id:195330)取值为 $x_0$ 的概率，即 $j(x_0) = P(X=x_0)$。这些具有正概率的点被称为[概率分布](@entry_id:146404)的**原子 (atoms)**。如果一个[随机变量](@entry_id:195330)的全部概率质量都集中在这些原子上，它就被称为**离散型[随机变量](@entry_id:195330)**。

关于这些不连续点（即原子）的集合，有一个重要的理论结果：任何 CDF 的[不连续点集](@entry_id:160308)合最多是**可数的 (at most countable)**。这是因为所有跳跃的高度之和不能超过 1。我们可以将所有跳跃点按其高度进行分类：高度大于 $\frac{1}{2}$ 的点、大于 $\frac{1}{3}$ 但不大于 $\frac{1}{2}$ 的点，以此类推。对于任何 $n \in \mathbb{N}$，高度大于 $1/n$ 的跳跃点个数不​​能超过 $n-1$ 个，否则它们的总高度将超过 1。因此，每个类别的点都是有限的，而所有不连续点的集合是这些有限集的并集，所以它最多是可数的。这意味着一个[随机变量](@entry_id:195330)不可能在不可数个点上具有正概率 [@problem_id:1416768]。然而，这个可数集可以是稠密的，例如，我们可以构造一个[随机变量](@entry_id:195330)，其原子恰好是所有的有理数 $\mathbb{Q}$ [@problem_id:1416768]。

#### 斜率与连续型[随机变量](@entry_id:195330)
CDF 的平滑增长部分则对应于概率质量的[连续分布](@entry_id:264735)。如果一个 CDF 在某个区间上是连续的，则[随机变量](@entry_id:195330)在该区间内取任何单一点的概率都为 0。如果 CDF 不仅连续，而且可微，其导数 $f_X(x) = F_X'(x)$ 就被称为**[概率密度函数](@entry_id:140610) (Probability Density Function, PDF)**。PDF 本身不是概率，但它描述了概率在[实数轴](@entry_id:147286)上[分布](@entry_id:182848)的“密度”。在区间 $[a, b]$ 上的概率可以通过对 PDF 进行积分得到：$P(a \le X \le b) = \int_a^b f_X(x) dx$。如果一个[随机变量](@entry_id:195330)的 CDF 处处连续，并且其概率可以通过 PDF 的积分给出，则称之为**绝对连续型[随机变量](@entry_id:195330)**。

#### 混合型[随机变量](@entry_id:195330)与[勒贝格分解](@entry_id:161722)
在许多实际应用中，[随机变量](@entry_id:195330)既非纯离散也非纯连续。例如，一个仪器的读数可能在某个范围内连续变化，但有一定概率饱和或取值为零。这类[随机变量](@entry_id:195330)被称为**混合型[随机变量](@entry_id:195330)**。

根据**[勒贝格分解定理](@entry_id:197665) (Lebesgue's Decomposition Theorem)**，任何 CDF 都可以唯一地分解为一个离散部分和一个连续部分的加权和（严格来说还有一个奇异连续部分，但在实际应用中极为罕见，此处从略）。
$$ F_X(x) = \alpha F_{ac}(x) + (1-\alpha) F_{d}(x) $$
这里，$F_{ac}(x)$ 是一个绝对连续型 CDF，$F_d(x)$ 是一个纯离散型 CDF（[阶梯函数](@entry_id:159192)），而 $\alpha \in [0, 1]$ 是连续部分的权重。权重 $1-\alpha$ 恰好等于 CDF 所有跳跃高度的总和。

考虑这样一个[混合模型](@entry_id:266571)：一个[随机变量](@entry_id:195330) $X$ 有 $p$ 的概率从 $[0, 1]$ 上的[均匀分布](@entry_id:194597)中抽取，有 $1-p$ 的概率取一个固定的常数值 $c \in [0, 1]$ [@problem_id:1416750]。其 CDF 可以通过[全概率定律](@entry_id:268479)得到：
$$ F_X(x) = p \cdot P(U \le x) + (1-p) \cdot P(c \le x) $$
其中 $U \sim \text{Uniform}(0, 1)$。$P(U \le x)$ 是[均匀分布](@entry_id:194597)的 CDF，而 $P(c \le x)$ 是一个在 $c$ 点从 0 跳到 1 的[阶梯函数](@entry_id:159192)。
经过分析，我们发现 $F_X(x)$ 在 $x=c$ 处有一个大小为 $1-p$ 的跳跃，这是其唯一的离散部分。其余部分则是平滑增长的。因此，这个混合 CDF 的分解结果是：
-   连续部分的权重 $\alpha = p$。
-   绝对连续部分的 CDF $F_{ac}(x)$ 就是[均匀分布](@entry_id:194597)的 CDF。
-   离散部分的权重 $1-\alpha = 1-p$。
-   离散部分的 CDF $F_d(x)$ 是一个在 $x=c$ 点取值为1的[随机变量](@entry_id:195330)的 CDF。

这个分解清晰地揭示了[混合随机变量](@entry_id:752027)的内在结构。

### 从 CDF 计算概率

CDF 的一个核心用途是计算[随机变量](@entry_id:195330)落在任意集合上的概率。对于任何一个 CDF $F(x)$，它都唯一确定了一个在实数集上的**[勒贝格-斯蒂尔杰斯测度](@entry_id:183409) (Lebesgue-Stieltjes measure)** $\mu_F$，该测度衡量了任意（可测）集合的“概率大小”。最基本的对应关系是：
$$ \mu_F((a, b]) = P(a  X \le b) = F(b) - F(a) $$
基于此，我们可以推导出计算其他类型区间概率的公式：
-   $P(X=a) = \mu_F(\{a\}) = F(a) - F(a^-)$，其中 $F(a^-) = \lim_{x \to a^-} F(x)$。
-   $P(X  a) = \mu_F((-\infty, a)) = F(a^-)$。
-   $P(X \ge a) = \mu_F([a, \infty)) = 1 - F(a^-)$。
-   $P(a \le X \le b) = \mu_F([a, b]) = F(b) - F(a^-)$。

让我们通过一个例子来实践。考虑以下 CDF [@problem_id:1416735]：
$$
F(x) = \begin{cases}
0  \text{if } x  0 \\
\frac{1}{4}  \text{if } 0 \le x  1 \\
\frac{1}{4} + \frac{x-1}{2}  \text{if } 1 \le x  2 \\
1  \text{if } x \ge 2
\end{cases}
$$
我们要计算集合 $S = [0, 1.5) \cup \{2\} \cup [3,4]$ 的测度 $\mu_F(S)$。由于这三个集合不相交，我们可以分别计算它们的测度然后相加。

1.  $\mu_F([0, 1.5)) = F(1.5^-) - F(0^-)$。
    $F(1.5^-) = F(1.5) = \frac{1}{4} + \frac{1.5-1}{2} = \frac{1}{2}$。
    $F(0^-) = \lim_{x \to 0^-} 0 = 0$。
    所以 $\mu_F([0, 1.5)) = \frac{1}{2} - 0 = \frac{1}{2}$。

2.  $\mu_F(\{2\}) = F(2) - F(2^-)$。
    $F(2) = 1$。
    $F(2^-) = \lim_{x \to 2^-} (\frac{1}{4} + \frac{x-1}{2}) = \frac{1}{4} + \frac{1}{2} = \frac{3}{4}$。
    所以 $\mu_F(\{2\}) = 1 - \frac{3}{4} = \frac{1}{4}$。这正是 $x=2$ 处的跳跃高度。

3.  $\mu_F([3, 4]) = F(4) - F(3^-)$。
    $F(4) = 1$，$F(3^-) = \lim_{x \to 3^-} 1 = 1$。
    所以 $\mu_F([3, 4]) = 1 - 1 = 0$。

最终，$\mu_F(S) = \frac{1}{2} + \frac{1}{4} + 0 = \frac{3}{4}$。这个过程展示了如何利用 CDF 对复杂集合进行概率度量。

### [变换随机变量](@entry_id:263513)：模拟的基础

CDF 在计算科学中最重要的应用之一是作为生成任意[分布](@entry_id:182848)随机数的理论基石。这依赖于两个互为逆运算的强大变换。

#### [概率积分变换](@entry_id:262799) (Probability Integral Transform, PIT)
PIT 是一个令人惊讶的优美定理：如果 $X$ 是一个[连续随机变量](@entry_id:166541)，其 CDF 为 $F_X(x)$，那么经过变换后的新[随机变量](@entry_id:195330) $U = F_X(X)$ 服从区间 $(0, 1)$ 上的**标准[均匀分布](@entry_id:194597)**。

这个定理的直观解释如下：$U$ 的值小于等于某个 $y \in (0,1)$ 的概率是 $P(U \le y) = P(F_X(X) \le y)$。由于 $F_X$ 是连续且单调的，我们可以对其求反函数，得到 $P(X \le F_X^{-1}(y))$。根据 CDF 的定义，这恰好等于 $F_X(F_X^{-1}(y)) = y$。$P(U \le y) = y$ 正是标准[均匀分布](@entry_id:194597)的 CDF。

考虑一个物理场景：一个量子系统的衰变时间 $T$ 服从均值为 $\tau$ 的指数分布，其 CDF 为 $F_T(t) = 1 - \exp(-t/\tau)$。一个信号处理器计算 $U=F_T(T)$。根据 PIT，无论 $\tau$ 是多少，$U$ 总是服从 $(0, 1)$ 上的[均匀分布](@entry_id:194597)。如果后续还有一个放大电路产生电压 $V = V_0 \sin(\frac{\pi}{2} U)$，我们就可以计算其期望输出电压 [@problem_id:1356792]：
$$ E[V] = E\left[V_0 \sin\left(\frac{\pi}{2} U\right)\right] = V_0 \int_0^1 \sin\left(\frac{\pi}{2} u\right) du = V_0 \left[-\frac{2}{\pi}\cos\left(\frac{\pi}{2} u\right)\right]_0^1 = \frac{2V_0}{\pi} $$
这个结果不依赖于具体的衰变时间常数 $\tau$，完全由变换后的[均匀分布](@entry_id:194597)决定。

#### [逆变换采样](@entry_id:139050) (Inverse Transform Sampling)
PIT 的逆向应用——[逆变换采样](@entry_id:139050)——是蒙特卡洛模拟中的核心技术。它指出，如果我们能生成一个服从标准[均匀分布](@entry_id:194597)的随机数 $U$，那么通过计算 $X = F^{-1}(U)$，我们就能得到一个服从目标分布 $F$ 的随机数 $X$。

这里的逆函数 $F^{-1}(y)$ 需要被推广为**[分位数函数](@entry_id:271351) (quantile function)**，以处理非严格单增的 CDF：
$$ F^{-1}(y) = \inf\{x \in \mathbb{R} : F(x) \ge y\} $$
这个定义寻找的是使得 CDF 值“首次”达到或超过 $y$ 的最小的 $x$ 值。它能正确处理 CDF 中的平坦[部分和](@entry_id:162077)跳跃。

作为一个具体示例，假设我们希望从以下 CDF 代表的[分布](@entry_id:182848)中采样 [@problem_id:1416756]：
$$
F(x) = \begin{cases}
0  \text{for } x  0 \\
\frac{1}{4} x^2  \text{for } 0 \le x  1 \\
\frac{3}{4}  \text{for } 1 \le x  2 \\
\dots
\end{cases}
$$
我们从[均匀分布](@entry_id:194597)中抽取一个随机数，例如 $u=0.2$。接下来，我们寻找满足 $F(x) \ge 0.2$ 的最小 $x$。
-   当 $0 \le x  1$ 时，$F(x)$ 的范围是 $[0, \frac{1}{4}) = [0, 0.25)$。我们的 $u=0.2$ 恰好落在这个范围内。
-   因此，我们求解 $\frac{1}{4}x^2 = 0.2$。
-   解得 $x^2 = 0.8$，由于 $x$ 在此区间内非负，所以 $x = \sqrt{0.8} \approx 0.8944$。

这样，我们就将一个均匀随机数 $u=0.2$ 转换为了一个服从[目标分布](@entry_id:634522)的随机数 $x \approx 0.8944$。

#### 经验 CDF 与 PIT 的局限性
在实践中，我们往往不知道真实的 CDF $F_X$，只能通过一组观测数据 $\{X_i\}_{i=1}^n$ 来估计它。最常用的估计是**[经验累积分布函数](@entry_id:167083) (Empirical CDF, ECDF)**：
$$ \hat{F}_n(x) = \frac{1}{n}\sum_{i=1}^n \mathbf{1}\{X_i \le x\} $$
即样本中小于等于 $x$ 的数据点的比例。

一个自然的问题是：如果我们用 ECDF 来进行[概率积分变换](@entry_id:262799)，即计算 $U_i = \hat{F}_n(X_i)$，这个新的样本 $\{U_i\}$ 会服从[均匀分布](@entry_id:194597)吗？[@problem_id:3183229]
-   **当 $X$ 是连续变量时**：样本中几乎不会有重复值。将样本排序为 $X_{(1)}  X_{(2)}  \dots  X_{(n)}$，我们发现 $\hat{F}_n(X_{(i)}) = i/n$。因此，变换后的样本 $\{U_i\}$ 只是集合 $\{\frac{1}{n}, \frac{2}{n}, \dots, \frac{n}{n}\}$ 的一个[排列](@entry_id:136432)。这是一个确定性的、离散的集合，并非[连续均匀分布](@entry_id:275979)。但当 $n$ 很大时，它确实在某种程度上“逼近”了[均匀分布](@entry_id:194597)。
-   **当 $X$ 是[离散变量](@entry_id:263628)时**：样本中会有大量重复值。所有等于同一个值 $v_k$ 的数据点，在经过 $\hat{F}_n$ 变换后，都会得到完全相同的 $U$ 值。这导致变换后的样本 $\{U_i\}$ 聚集在少数几个值上，与[连续均匀分布](@entry_id:275979)的形态相去甚远。

这一分析揭示了理论与实践之间的重要差异，并解释了为什么对离散数据直接应用某些基于连续分布假设的统计方法（如某些版本的[KS检验](@entry_id:751068)）可能会产生误导性结果。

### [分布](@entry_id:182848)的收敛

最后，CDF 在描述[随机变量](@entry_id:195330)序列的极限行为时也扮演着核心角色。一个[随机变量](@entry_id:195330)序列 $\{X_n\}$ **[依分布收敛](@entry_id:275544) (converge in distribution)** 到[随机变量](@entry_id:195330) $X$，是指它们的 CDF 序列 $\{F_n(x)\}$ 在 $F(x)$ 的所有连续点上都逐点收敛到 $F(x)$。

考虑一个例子 [@problem_id:1416763]：对每个正整数 $n$，[随机变量](@entry_id:195330) $X_n$ 从集合 $\{\frac{1}{n}, \frac{2}{n}, \dots, \frac{n}{n}\}$ 中取值，且取值为 $\frac{k}{n}$ 的概率正比于 $k$。
首先，可以算出其[概率质量函数](@entry_id:265484)为 $P(X_n = k/n) = \frac{2k}{n(n+1)}$。对于 $x \in [0, 1)$，其 CDF 为：
$$ F_n(x) = P(X_n \le x) = \sum_{k=1}^{\lfloor nx \rfloor} \frac{2k}{n(n+1)} = \frac{2}{n(n+1)} \frac{\lfloor nx \rfloor(\lfloor nx \rfloor + 1)}{2} = \frac{\lfloor nx \rfloor}{n} \cdot \frac{\lfloor nx \rfloor + 1}{n+1} $$
现在我们考察当 $n \to \infty$ 时 $F_n(x)$ 的极限。由于 $\lfloor nx \rfloor$ 与 $nx$ 相差不到 1，我们有 $\frac{\lfloor nx \rfloor}{n} \to x$。类似地，$\frac{\lfloor nx \rfloor + 1}{n+1} \to x$。
因此，对于任意 $x \in [0, 1)$，
$$ \lim_{n \to \infty} F_n(x) = x \cdot x = x^2 $$
结合边界情况，我们发现这列[离散随机变量](@entry_id:163471)的 CDF 序列[逐点收敛](@entry_id:145914)到了一个连续的 CDF：
$$ F(x) = \begin{cases} 0,  x0 \\ x^2,  0\le x\le 1 \\ 1,  x>1 \end{cases} $$
这个过程优美地展示了[离散分布](@entry_id:193344)如何通过极限过程逼近连续分布，这是概率论中心极限定理等许多核心结果的基石。
